我们的网站使用 Hugo 静态网站生成器和 Mainroad 主题构建。这个演示允许您在决定使用之前查看基于 Mainroad 的网站是什么样子。请记住，当前的演示展示了基础内容，仅仅是 Hugo 和 Mainroad 组合所能实现的一小部分。

关于作者

Mainroad 是一个响应式、简洁、干净且以内容为中心的 Hugo 主题，基于 MH Magazine lite WordPress 主题。

主要特点：

  • 响应式设计
  • 主菜单和次菜单
  • 小部件化的侧边栏
  • 翻译。超过 15 种语言并在不断增加
  • 可配置的主题设置（侧边栏位置、作者框、文章导航、高亮颜色）通过 config.toml
  • Hugo 内部模板（Open Graph、Schema、Twitter Cards、Disqus、Google Analytics）
  • 广泛的跨浏览器兼容性
    • 桌面：IE11+、Chrome、Firefox、Safari
    • 移动端：Android 浏览器（Android 4.4+）、Safari（iOS 7+）、Google Chrome、Opera mini
  • 自定义 Google 字体支持、MathJax、目录、SVG 图标等等……

在 GitHub 上了解更多。Mainroad 主题是根据 GPLv2 许可证 发布的。

关于本博客

Hugo 是一个用 Go 编写的静态 HTML 和 CSS 网站生成器。它针对速度、易用性和可配置性进行了优化。Hugo 将一个包含内容和模板的目录渲染成一个完整的 HTML 网站。凭借其惊人的速度和灵活性，Hugo 让构建网站重新变得有趣。

在 GitHub 上了解更多。完整的文档可在 Hugo 文档 中找到。
