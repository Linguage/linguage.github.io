Heterogeneous Graph Attention Network for Rail Fastener Looseness Detection Using Distributed Acoustic Sensing

2025-01-01

摘要

本文针对铁路扣件松动检测问题，提出一种融合分布式声学传感（DAS）与加速度计数据的异构图注意力网络模型 FusionHGAT。该模型将监测场景抽象为包含三类节点（扣件、DAS测量点、加速度计）的异构图，通过1D-CNN提取局部时序特征，经Transformer编码器嵌入全局时序依赖，再利用多层图注意力网络（GAT）融合多源空间信息，最终以节点分类任务输出每个扣件的松动状态。实验在新加坡武吉班让地铁站在役轨道上开展，共采集182次锤击激励数据，涵盖6种扣件松紧组合工况。测试结果表明，FusionHGAT在多扣件松动检测任务中实现了100%准确率和1.0 AUC值。消融实验和超参数分析进一步验证了各模块的必要性及关键参数的最优配置。此外，研究系统评估了传感器数量、位置和感知距离对检测精度的影响，为工程部署提供了经济可行的传感器优化布置策略。

研究背景

铁路扣件是连接钢轨与轨枕的关键部件，负责维持轨距并减缓轮轨动态交互力。在高速运营、环境侵蚀和长期机械振动的综合作用下，扣件易发生松动乃至断裂，直接威胁行车安全。传统人工巡检方式耗时耗力且容易遗漏，随着铁路网络规模的持续扩张，已难以满足实际需求。

现有自动化检测方法主要分为三类。基于图像的方法从早期像素级特征发展至 Fastener R-CNN、YOLO 等深度学习框架，但依赖高分辨率专用设备，计算成本较高。基于振动的方法利用加速度计捕获轨道动态响应，结合 FFT、小波变换及1D-CNN 进行特征分析，在实验室环境中效果良好，但大规模部署成本较高。基于 DAS 的方法可从单一端点沿数十公里光纤连续监测高频振动，成本效益突出，但信噪比相对较低，检测精度尚不及加速度计方法。

上述方法在多源数据融合与空间关系建模方面均存在明显不足。扣件与传感器的非均匀、不规则空间分布难以用传统欧氏几何表达。图神经网络（GNN）专为非欧几里得图结构数据设计，已在交通预测、分子属性预测及结构健康监测等领域展现出强大的空间建模能力，为本研究提供了方法论基础。

核心内容

异构图构建

监测场景被抽象为异构图 G = (N, E, W)，包含三类节点：扣件节点（nF）、DAS测量点节点（nD）和加速度计节点（nA），共定义9种边类型（FD、FA、DF、DA、AF、AD、FF、DD、AA），支持不同来源信息的双向消息传递。节点间连通性由距离阈值 dt 控制，仅当两节点物理距离小于 dt 时建立边连接。边权重采用 Minkowski 距离的倒数计算，并设置最大权重 wmax = 100 以防数值溢出。实验中邻接矩阵规模为13×13，涵盖5个加速度计、2个DAS测量点和6个扣件节点。扣件松动检测被建模为节点二分类任务，节点向量 [0,1] 表示紧固状态，[1,0] 表示松动状态。

FusionHGAT 模型架构

模型由三个串联模块构成：

第一模块为1D-CNN特征提取器，针对DAS和加速度计的时间序列信号分别设计独立的4层卷积结构，包含卷积层、最大池化层和批归一化层，将原始信号转化为高维局部特征表示。

第二模块为Transformer编码器，仅使用编码器部分（舍弃用于序列生成的解码器），配置2层编码器、4个注意力头，嵌入维度为12，用于捕捉CNN无法建模的全局时序依赖关系，进一步精化特征表示。

第三模块为多层图注意力网络（GAT），配置2层图注意力层（GAL），自注意力头数为8，节点特征维度从12逐层降至2。GAT通过 LeakyReLU 激活的注意力系数动态加权邻居节点特征，最终经 Softmax 层输出每个扣件节点的松动概率。

实验设置

实验在新加坡武吉班让地铁站在役轨道开展，扣件间距0.7 m，部署1根DAS光纤电缆（测量点间隔1.25 m）和5个加速度计（竖向3个、水平2个）。采用冲击锤激励，共设计6种工况（全紧固至多点交替松动），累计锤击182次。每次激励产生17通道数据（2路DAS + 15路三轴加速度计）。训练/验证/测试集比例为70%/15%/15%，使用二元交叉熵损失函数，学习率0.001，Dropout率0.2。

消融实验

消融实验系统比较了不同模块替换方案的性能。完整模型（1D-CNN + Transformer + GAT）在测试集上均值准确率达99.82%。去除特征提取层（Comp. #2）导致模型将所有样本预测为紧固状态，AUC仅为0.435，完全丧失松动检测能力。以 LSTM 替代1D-CNN（Comp. #1）准确率降至96.96%。去除 Transformer（Comp. #3）准确率略降至98.77%但出现明显过拟合。以 GCN 替代 GAT（Comp. #4）准确率降至93.69%，因 GCN 缺乏自适应加权机制。以全连接层替代 GAT（Comp. #5）准确率最低（93.04%）且过拟合明显。

超参数与鲁棒性分析

超参数分析表明，GAT层数以2层最优（超过2层性能下降，达5~6层时完全失去松动检测能力）；注意力头数8个最优；特征嵌入维度12最优；连接阈值0.8最优；距离度量以 L2 范数（p=2）表现最佳，准确率99.821%。噪声鲁棒性评估显示，在噪信比（NSR）不超过40%时模型准确率保持95%以上，NSR高达70%时仍维持约90%的准确率。

与其他 GNN 模型的比较

FusionHGAT（99.82%，推理时间1.612 ms）在测试集准确率上优于 GraphSAGE（96.45%）、GIN（97.12%）、EGNN（99.03%）、PNA（98.77%）和 GSAT（98.15%），推理时间同样具有竞争力。

可扩展性与传感器优化布置

推理时间随监测长度呈对数增长（而非线性增长），拟合方程 y = a − b·ln(x + c)，R² = 0.971，监测范围达1000米时推理时间仍低于300毫秒。传感器数量分析表明，仅使用 DAS 时平均准确率97.14%，增加1个加速度计后提升至99.226%，继续增加收益递减。传感器位置以轨承点附近的轨翼处最优（99.383%）。传感距离增大会导致两类传感器的检测精度均值下降、稳定性降低，且加速度计在各距离下整体优于 DAS。

关键概念

FusionHGAT：本文提出的融合异构图注意力网络，核心框架为1D-CNN + Transformer + GAT，用于多源传感数据融合的铁路扣件松动检测

分布式声学传感（DAS）：基于相位敏感光时域反射（Φ-OTDR）原理的光纤传感技术，可从单一端点沿数十公里光纤连续测量高频动态振动，成本低、覆盖范围广

异构图（Heterogeneous Graph）：包含多种类型节点和边的图结构，本文用于容纳扣件、DAS测量点和加速度计三类异质实体，建模其非规则空间关系

图注意力网络（GAT）：由 Veličković 等人于2018年提出，通过可学习的注意力系数对邻居节点特征进行自适应加权聚合，优于不具备自适应加权的图卷积网络（GCN）

Minkowski 距离：用于计算边权重的空间距离度量，本文以其倒数量化节点间交互强度，L2 范数（欧氏距离，p=2）表现最优

节点分类任务：将扣件松动检测问题形式化为图上的节点二分类问题，每个扣件节点被分类为紧固或松动两种状态

噪信比（NSR）：引入白噪声评估模型鲁棒性的指标，定义为噪声幅度系数与原始信号均方根值之比

消融实验：通过逐一移除或替换模型中的各功能模块，定量评估每个模块对整体性能贡献的实验方法

模型蒸馏（Model Distillation）：作者提出的未来优化方向，通过将复杂模型的知识迁移到轻量化模型，降低推理所需计算资源

主要贡献

  1. 异构图建模框架：首次将铁路扣件监测场景建模为包含三类异质节点和9种边类型的异构图，有效捕捉传感器与结构部件之间复杂、不规则的非欧空间关系，突破了传统欧氏几何描述的局限。

  2. FusionHGAT 模型：提出集1D-CNN特征提取、Transformer时序嵌入和多层GAT空间融合于一体的端到端多源数据融合模型，在现场锤击实验中实现了100%检测准确率（F1-score = 1.0，AUC = 1.0），优于 GraphSAGE、GIN、EGNN、PNA、GSAT 等现有图神经网络方法。

  3. 传感器优化布置策略：系统量化了传感器数量、位置和感知距离对检测精度的影响，提出仅使用 DAS 即可达到97.14%准确率、增加1个加速度计可提升至99.226%的渐进式部署建议，为工程应用中成本与性能的权衡提供了量化依据。

  4. 可扩展性验证：通过虚拟节点仿真证明推理时间随监测长度呈对数增长，1000米范围内推理时间低于300毫秒，且单公里推理时间不超过1秒，验证了模型在大规模铁路网络中的实时应用潜力。

  5. 鲁棒性评估：在NSR不超过40%时准确率维持在95%以上，NSR高达70%时仍约90%，证明模型在真实工程噪声环境下具有较强的抗干扰能力。

局限与展望

当前局限

  • 与传统基于频率域的模型相比，FusionHGAT 所需计算资源更多，可能需要引入模型蒸馏等轻量化技术提升部署效率。
  • 预处理模块相对简单，目前仅在受控锤击测试场景下得到验证，对实际列车荷载激励、不同轨道类型及复杂环境条件的适用性尚需进一步验证。
  • 实验规模相对有限（6个扣件、182次激励），需在更多样化的真实场景中开展大规模测试以增强结论的普适性。
  • 当前采用的信号预处理方法较为基础，复杂噪声场景下可能需要更先进的降噪策略。

未来展望

  • 引入频率分析、小波变换及高级去噪算法提升信号质量和特征表达能力。
  • 研究图模型的剪枝策略以提升大规模监测的计算效率，并探索单一大图与多个分段小图的权衡方案。
  • 扩展模型对不同环境条件、轨道几何形状和扣件类型的泛化能力。
  • 引入半监督学习方法，利用人工巡检报告或部分标注数据集进行弱监督训练，应对实际运营中标注数据不足的挑战。
  • 探索基于列车时刻表或音频触发的自动化信号采集方案，降低人工介入需求。
