The effect of fastener clip fatigue for high-speed railway on vehicle-track dynamic interaction：Numerical analysis

2024-01-01

摘要

本文针对高速铁路扣件弹条在列车循环荷载作用下的疲劳损伤演化及其对车辆-轨道（VT）系统动力相互作用的反馈影响展开系统性数值研究。研究以CRTS III型无砟轨道为对象，建立了兼顾高效计算与长期演化的集成分析框架：采用时域格林函数法与直接积分法的混合求解策略提升计算效率；以雨流计数法统计应力循环、结合S-N曲线与Goodman修正及Palmgren-Miner线性累积损伤理论预测弹条疲劳寿命；利用Karhunen-Loève展开（KLE）将扣件抗拉强度的空间变异性建模为随机场，在概率框架下评估系统长期动力性能。主要结论表明：列车循环次数增加导致车辆-轨道系统动力响应显著恶化，扣件损伤遵循三阶段规律；非均匀扣件系统的疲劳损伤大于均匀情况；轨道不平顺的短波分量是导致弹条疲劳损伤的最主要因素，其影响大于行车速度；保持轨道几何平顺性、提高扣件均匀性及适当控制行车速度是延长扣件系统服役寿命的可行工程措施。

研究背景

铁路扣件系统是连接钢轨与轨道板的关键结构部件，承担着传递列车荷载、维持轨道几何形态的核心功能。在列车高频循环动态荷载的长期作用下，扣件弹条不可避免地发生疲劳损伤乃至断裂，其直接后果是扣压力下降、导致系统动力失稳，进而威胁列车运营安全乃至引发脱轨风险。

扣件的物理和力学性能因材料特性、安装工艺及预紧力等因素表现出随机性与空间变异性。不均匀的线路条件（如随机轨道不平顺）进一步加剧扣件性能的空间非均匀性，刚度的不规则分布反过来增大轮轨冲击，形成恶性循环。

在既有研究方面，相关工作主要沿两条路径展开：其一是VT动力学分析模型的高效精确求解方法研究；其二是将外部服务条件（如结构疲劳损伤、温度变化、水分迁移等）耦合进VT动力学模型的研究。针对弹条疲劳，前人已通过实验结合有限元（如ANSYS、Abaqus）评估了弹条的疲劳寿命与失效机制，普遍采用雨流计数法与S-N曲线作为分析工具。然而，这些研究或仅关注瞬态动力行为而忽视长期演化，或采用解耦方式无法实现完全耦合求解，且均未从车辆-轨道交互视角系统考察扣件疲劳损伤对VT动力系统的反向影响。此外，已有研究引入随机场理论分析扣件参数变异性对VT动力行为的影响，但忽略了扣件疲劳损伤。上述研究空白共同指向本文的核心问题：考虑扣件弹条长期疲劳损伤的VT随机动力模型尚未建立，缺乏相应的疲劳损伤演化分析与长期动力性能评估方法。

核心内容

车辆-轨道动力相互作用模型

车辆建模为38自由度多刚体系统，包含1个车体、2个转向架和4个轮对，各刚体具有纵向、横向、垂向、偏转、侧滚和点头共6个自由度（不考虑牵引制动时轮对纵向运动可忽略，简化为5个自由度）。钢轨采用Bernoulli-Euler梁单元（垂向、横向运动）与一维杆单元（纵向、扭转）组合建模。混凝土轨道板和底座采用Reissner-Mindlin板单元建模。车轮-钢轨相互作用通过非线性Hertz接触理论和Kalker蠕滑理论实现耦合。

为提升计算效率，模型采用混合求解策略：将系统分为高频车辆-钢轨子模型（时域直接积分法，时间步长3×10⁻⁴ s）和低频轨道板-底座子模型（时域格林函数法）。两个子模型通过非线性伪扣件力维持耦合兼容。格林函数通过傅里叶变换推导，卷积积分采用Mazilu提出的快速数值表达式实现。该模型与Zhai经典车辆-轨道耦合动力学模型进行对比验证，在车体垂向加速度、轮轨垂向力及扣件力的时间历程与功率谱密度上均吻合良好，在60–100 Hz频段存在因数值方案差异引起的小幅偏差。

扣件系统建模为弹簧-阻尼器，具有非线性刚度特征：压缩状态下刚度为轨道垫板刚度的一半，拉伸状态下刚度由弹夹预锚固荷载（25 kN）与钢轨-轨道板相对位移决定，且弹夹在拉伸状态下不提供阻尼效应。

疲劳损伤模型

以WJ-8型扣件弹片（材料60Si2MnA钢）为例，S-N曲线表达式为 lg N = 39.5953 − 11.8436 · lg Sₐ，其中 Sₐ 为疲劳应力幅。现场统计表明弹条最易在弧形接头处（危险区域）发生损伤，该处von Mises应力与轨道板相对位移近似呈线性关系：Sᶜ = 1580 + 162.8 · ΔXf（MPa）。

应力循环采用雨流计数法统计。考虑到螺栓预紧引入的初始应力，采用Goodman模型对基于应力比R = −1的S-N曲线进行平均应力修正。累积疲劳损伤遵循Palmgren-Miner线性叠加理论：D = Σ(nᵢ/Nᵢ)。损伤后的扣件刚度和阻尼系数按线性退化规律处理，各方向刚度随损伤量D退化为(1 − D)·kf(0)，完全损伤的扣件仅向钢轨提供向上支撑力。含损伤扣件的非线性VT系统采用迭代法求解，迭代过程包括更新损伤值、求解系统响应、更新非线性系数和扣件力，直至两子系统动力响应收敛。为兼顾效率，模型仅在动力分析前更新疲劳损伤引起的刚度变化，结合基于S-N曲线的循环跳跃插值算法，每个样本仅需20个循环跳跃步即可完成弹条全疲劳损伤过程的模拟。

随机场建模

扣件弹片抗拉强度沿轨道纵向的空间变异性通过Karhunen-Loève展开（KLE）建模为高斯随机场，均值1800 MPa，变异系数0.01。空间相关结构采用指数核函数描述，波动距离取为轨道板长度。截断项数M = 100，综合计算效率与精度（通过Kullback-Leibler散度收敛性验证），最终选取200个随机场样本。

数值算例主要发现

数值算例基于CRTS III型轨道系统，对比了不同列车循环次数（4×10⁶至20×10⁶次）、行车速度（300/350/400 km/h）、列车类型（CRH380与ICE）以及轨道不平顺波长（短波与中长波）的影响。

随列车循环次数增加，车体加速度极值持续增大（相对4×10⁶次工况，在8、12、16、20×10⁶次时分别增大70.3%、89.1%、137.4%、163.7%），钢轨位移均值和标准差均显著增大，在20×10⁶次循环下最大位移可达3.5 mm，超出规范动态位移阈值（2.0 mm）75%。钢轨位移概率密度函数（PDF）在较多循环次数下出现双峰现象，第二峰约出现在2.75 mm处，推测由扣件局部过度损伤引起。扣件损伤均值从10×10⁶至20×10⁶次循环的增量（约0.27）远大于10×10⁶次以前的增量（约0.086），呈现非线性加速退化特征。

行车速度提高加剧损伤：350 km/h下损伤大于0.9的扣件比例为15%，400 km/h下增至19.8%。非均匀工况（350 km/h）下损伤大于0.9的比例约为均匀工况的4倍。短波长轨道不平顺激励下，损伤大于0.9的扣件比例（48.3%）约为中长波工况（15.1%）的2.2倍，是影响扣件损伤最主要的因素，其影响大于行车速度，行车速度的影响又大于列车轴重差异。

关键概念

车辆-轨道动力相互作用：车辆系统（车体、转向架、轮对）与轨道结构（钢轨、扣件、轨道板、底座）之间通过轮轨接触力和扣件力实现耦合的动力学系统，本文研究其在扣件疲劳损伤演化背景下的长期行为。

扣件弹条（弹片）疲劳：扣件中弹性夹片在列车循环荷载作用下的累积损伤过程，集中发生在弧形接头危险区域，由轨道不平顺退化和车轮多边形磨耗引起的高频应力循环驱动。

雨流计数法：用于统计随机应力时程中疲劳循环次数及应力幅值的方法，是本文疲劳损伤计算的基础工具之一。

S-N曲线：描述材料疲劳应力幅（S）与疲劳寿命（N）关系的经验曲线，本文针对60Si2MnA钢弹条采用对数线性形式：lg N = 39.5953 − 11.8436 · lg Sₐ。

Goodman模型：用于将含平均应力的实际疲劳条件转化为等效对称循环（R = −1）条件的平均应力修正方法，以考虑螺栓预紧引入的初始应力影响。

Palmgren-Miner线性累积损伤理论：将各应力幅值水平下的循环损伤线性叠加以预测总累积损伤的方法，公式为 D = Σ(nᵢ/Nᵢ)。

时域格林函数法：利用系统导纳在频域中求解格林函数、再通过卷积积分获得时域响应的高效动力求解方法，本文用于低频轨道板-底座子模型的求解以提升计算效率。

Karhunen-Loève展开（KLE）：将随机场分解为均值场与由一组独立随机变量加权正交特征函数级数表示的零均值随机场之和，用于高效表征扣件抗拉强度的空间变异性。

指数核函数：描述随机场空间相关结构的协方差函数形式，C_B(u, u’) = exp(−|u−u’|/L)，其中L为波动距离（本文取为轨道板长度）。

Kullback-Leibler散度（KLD）：衡量模拟概率分布与理论概率分布之间差异的信息论指标，用于验证KLE随机场模拟的充分性。

三阶段损伤规律：本文发现列车引起的扣件累积损伤概率分布在损伤指标小于0.2、介于0.2至0.9之间、大于0.9三个阶段呈现不同特征，且损伤状态主要集中于低损伤和高损伤（接近失效）两端。

循环跳跃插值算法：为大幅提升随机分析计算效率而引入的策略，通过在S-N曲线基础上跳跃式更新损伤状态（仅需20个跳跃步即可模拟全疲劳寿命过程），避免逐循环更新带来的巨大计算代价。

CRTS III型无砟轨道：中国铁路轨道系统III型板式无砟轨道，由钢轨、扣件、混凝土轨道板、填充层和混凝土底座组成，是本文数值算例的研究对象。

主要贡献

  1. 首次建立了考虑扣件弹条长期疲劳损伤的VT随机动力耦合模型，填补了从车辆-轨道交互角度分析扣件疲劳损伤对VT系统长期性能影响方法的空白，实现了疲劳损伤演化与系统动力响应的双向耦合分析。

  2. 提出了兼顾精度与效率的集成分析框架，将时域格林函数混合求解方法、雨流计数-S-N曲线-Goodman-Miner损伤链以及KLE随机场建模有机集成，结合循环跳跃插值算法，使得每个随机样本仅需20个跳跃步即可完成全寿命模拟，大幅降低了随机分析的计算代价。

  3. 揭示了扣件损伤演化的三阶段规律与非线性加速退化特征，定量表明损伤均值在10×10⁶至20×10⁶次循环阶段的增量（约0.27）远大于前期增量（约0.086），为维护决策的时机选择提供了科学依据。

  4. 定量阐明了空间非均匀性的必然性与危害性，指出即使初始条件均匀，多次循环后轮轨接触的随机性也会导致扣件系统出现空间变异性，且非均匀扣件系统的疲劳损伤大于均匀情况，同时揭示了钢轨位移PDF双峰现象的成因机理假设。

  5. 系统比较了多种影响因素的重要性排序，明确给出轨道不平顺短波分量为最主要影响因素，其次为行车速度，轴重差异影响最小，为高速铁路扣件系统维护优先级制定提供了量化依据，并提出了保持轨道几何平顺性、提高扣件均匀性和控制行车速度三项工程建议。

局限与展望

模型局限性：

  • S-N曲线仅基于应力比R = −1获得，未考虑温度、湿度等环境条件对疲劳损伤的影响，模型适用范围存在限定。
  • 扣件以下混凝土轨道结构被假设为无损伤线弹性体，这是应用格林函数法的关键前提，在实际中轨道板随服役年限也会发生退化，该假设可能带来一定误差。
  • 在不考虑牵引和制动的条件下，轮对纵向运动被忽略，轮对简化为5个自由度，不适用于需考虑制动工况的场景。
  • 现有商业软件的Co-simulation方法无法实现完全耦合求解；将弹条振动模态耦合进VT动力学的方法也难以适用于大规模损伤演化分析。

潜在展望方向：

  • 可在概率框架中引入环境因素（温度、湿度）对疲劳损伤的影响，进一步完善全寿命预测模型。
  • 可通过预设更多循环跳跃步获取更完整的全寿命周期响应规律，并结合现场监测数据对模型进行率定与验证。
  • 研究建议将轨道几何状态的定期养护和扣件均匀性控制纳入高速铁路维护体系，为制定基于损伤阈值的预防性维护策略提供进一步的研究基础。

相关工作

本文在引言和方法论部分明确讨论了以下几项既有工作：

  • Zhai的车辆-轨道耦合动力学模型：该模型轨道系统采用模态叠加法建立，使用Zhai显式积分方法求解，是本领域最常被用于他人模型验证的基准模型，本文将其作为VT动力模型的对比验证标准，两者在车体垂向加速度、轮轨垂向力和扣件力等关键响应上吻合良好。

  • Yuan等人的VT动力学研究：建立了车辆-轨道动力学模型以研究轨道不平顺引起高频振动条件下扣件的力学性能，但仅考虑瞬态动力行为，未考虑长期损伤演化。

  • Gao等人的精细有限元模型研究：建立了精细有限元模型分析W1型扣件弹条在高速铁路中的失效机制，同样仅考虑瞬态行为。

  • Diego等人的疲劳寿命评估研究：通过实验结合ANSYS评估扣件弹簧夹疲劳寿命，采用S-N曲线的概率描述方法，为本文S-N曲线方法的应用提供了依据。

  • Yan等人的弹条固有频率研究：发现弹条疲劳寿命受弹条二阶固有频率共振的显著影响。

  • Xiao等人的解耦分析方法：采用VT动力模型提取扣件力后再在Abaqus中进行疲劳分析，指出弹条疲劳或断裂会导致扣压力下降从而引起系统动力失稳，但该方法为解耦方式，无法实现完全耦合求解。

  • Liu等人的随机场研究：引入随机场理论研究扣件参数变异性对VT动力行为的影响，但忽略了扣件疲劳损伤，是本文随机场框架的重要参照与待弥补的研究空白之一。

  • Mazilu提出的卷积积分数值表达式：为格林函数法中卷积积分的快速精确实现提供了数值方案，本文直接采用该方法实现轨道板-底座子模型的高效求解。
