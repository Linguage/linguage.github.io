Study on failure analysis and reliability optimization of high-speed railway fastener system

2025-01-01

摘要

本文以W300高速铁路扣件系统为研究对象，系统揭示了弹条和底板垫板两类主要易损部件的失效机理，并在此基础上开展结构与性能优化，最终通过可靠性模型对比优化前后的系统服役寿命。研究表明：弹条失效的根本原因是轮轨磨耗激励频率（540～630 Hz）与弹条第三阶固有频率（520～640 Hz）高度吻合，引发共振疲劳断裂；底板垫板则因聚合物材料在轮轨疲劳载荷下逐渐固化、分子间交联键密度增加，导致服役刚度随运营时间线性上升，形成刚度老化失效。针对上述失效模式，作者分别开发了高频弹条（结构优化使固有频率远离激励频带）和新型磁流变弹性体（MRE）底板垫板（可通过外部磁场实时调节刚度）。基于串联系统可靠性理论建立的综合评估模型表明，在轮轨激励、磨耗幅值和环境温度三重作用下，优化后高速铁路扣件系统的综合服役寿命提升3.27倍，具有更高的轨道适应性和运营可靠性。

研究背景

高速铁路扣件系统是连接钢轨与轨枕的关键结构单元，承担维持轨道动态几何参数、减振降噪及绝缘等多重功能，对行车安全、平稳性和旅客舒适性至关重要。随着高速铁路运营年限增加，两类典型失效问题日益突出：

其一，异常轮轨磨耗（车轮18～22阶多边形、轨道波长120～160 mm）在不同运行速度下产生不均匀高频激励，诱发弹条损伤和断裂，造成安全隐患。京沪、武广高铁现场统计表明，轮轨磨耗高频激励在弹条失效中占比高达98.1%，而线路平纵面形态及弹条内外侧位置与损伤无统计关联。

其二，底板垫板作为高分子聚合物部件，在温度（-50～60°C）、频率（0～2000 Hz）、载荷（0～200 kN）三维复合服役环境下，受疲劳载荷驱动的材料固化影响，刚度随服役年限线性增大，改变轨道结构受力状态并加剧振动响应。

既有研究已在弹条振动疲劳机制、底板垫板老化特性等单一维度积累了一定成果，但如何系统性改善高铁扣件疲劳失效、量化铁路多因素对扣件系统疲劳可靠性的综合影响，仍缺乏深入研究。本文即针对这一空白展开工作，体现了产学研合作背景（上海工程技术大学、华东交通大学及洛阳隋日橡塑科技、汉德潘德罗公司联合参与）。

核心内容

弹条失效机理分析

通过中国铁道科学研究院（CRCC）支持的大规模现场调查，结合模态测试与高频疲劳试验，确认W300扣件ω型弹条的失效模式为跟部完全断裂。弹条直径15 mm，材料为38Si7弹簧钢，失效断裂导致夹紧力完全丧失。

模态试验表明，弹条第三阶固有频率范围为520～640 Hz；在线振动测试显示，列车以300 km/h运行时弹条垂向峰值频率分别为522 Hz和605 Hz，与第三阶固有频率高度重叠，构成共振条件。高频疲劳试验机验证：按标准扭矩（250 N·m）安装的弹条试样在持续加载2小时5分钟后即发生断裂，从实验室层面确认了共振损伤机制。

弹条寿命预测模型以激励频率f和轮轨磨耗幅值λ为输入，整合轮轨磨耗幅值与弹条振动幅值的换算关系、动态应变公式、胡克定律及Goodman平均应力修正，结合拟合的S-N曲线（log N = 11.43 − 1.75 log S，R²=0.959）构建综合预测模型。分析表明，磨耗幅值越大、激励频率越接近600 Hz共振点，弹条疲劳寿命越短。

底板垫板失效机理分析

从20余条采用W300扣件的高速铁路中抽取240块底板垫板进行静刚度测试。设计刚度为23±3 kN/mm，实测数据显示：43.4%的垫板刚度已升至23～26 kN/mm，81.5%不超过32 kN/mm，全部样本不超过40 kN/mm，但均未出现开裂或挤出等典型损伤破坏。服役刚度随运营时间的拟合关系为：Kᵤ = 2.1×10⁴h + 21.5（线性增长），成因在于聚合物材料在轮轨疲劳载荷下逐渐固化，分子间交联键密度持续增加。

进一步在温频相关动态试验机上，在-40～60°C六个温度点与1～6×10ⁿ次疲劳循环的耦合条件下测试刚度演变规律，建立综合刚度模型：K_T&N = 8.2×10⁻⁷(1.02√(T−23)+23 − 0.41)N + 22.3。在固定温度下刚度随循环次数近似线性增长；在固定循环次数下刚度随温度呈先降后升的抛物线规律。极端温度下偏离设计值尤为显著：-40°C时刚度为设计值的2.8倍，60°C时为设计值的1.8倍；-40°C对应弹性材料的玻璃化转变温度，材料变硬变脆，可靠性最低。

弹条结构优化（高频弹条）

优化原则为在保证安装性能前提下使固有频率远离激励频带。优化后弹条长度从175 mm增至205 mm，高度从59 mm降至50 mm，质量从825 g降至790 g。结构调整使弹条模态由原来以竖向运动为主转变为以纵向（沿铁路方向）运动为主，跟部附近应力集中程度显著降低，最大等效应力降低50 MPa至1280 MPa。锤击试验验证两阶固有频率分别提升43.1%和40.9%，频响幅值同步降低。高频疲劳试验中，优化弹条运行至6,695,400次循环（等效2.69小时）仍未损坏，试验因保护设备安全而主动终止；与原弹条失效寿命相比，疲劳寿命提升31.9%。

底板垫板性能优化（MRE垫板）

引入磁流变弹性体（MRE）智能材料，其工作原理是：在外加磁场下，聚合物基体中的微米级软磁颗粒形成链状结构，改变材料刚度与阻尼；撤去磁场后材料恢复原始状态，具有可逆调控特性。MRE采用BASF公司CN、CD、SQ三种羰基铁粉按炼胶→混炼→磁场预制→硫化成型工序制备，最优试件MRE-2#（CN型羰基铁粉，添加量50%）综合性能最佳。

动态测试表明，随磁场（电流）增大，储能模量持续增大，陡降峰频率从120 Hz向高频方向移动。静态测试表明，通入5 A电流时MRE垫静刚度较无磁场时提升47.8%。由于现场实测7～8年服役后刚度老化增幅约40%，MRE垫47.8%的可调刚度范围可完全覆盖这一老化幅度，从而实现运营期间实时补偿刚度退化的工程目标；磁粉与橡胶基体的分子键混合还在一定程度上提高了耐温性，降低了刚度对温度的敏感性。

可靠性建模与对比分析

基于串联系统可靠性理论，将弹条和底板垫板视为串联单元：R_S(t) = R_a1(t) × R_a2(t)。弹条寿命N_a1和底板垫寿命N_a2均假设服从正态分布，标准差取均值的10%（σ=0.1μ）。

三种工况下的优化效果：

  • 激励频率（500～700 Hz）：优化后可靠性在全频段均有提升，最大增幅在600 Hz共振点处，寿命提升3.33倍
  • 轮轨磨耗幅值（0～0.06 mm）：优化前最低可靠性出现在0.06 mm处（振动增加6 dB导致裂纹），优化后在0.02 mm处可靠性提升最大，达3.5倍
  • 环境温度（-40～60°C）：优化前最低可靠性出现在-40°C（玻璃化转变点），最高在20°C（高弹态）；优化后在-40°C处提升幅度最大，达3倍

综合三种工况，优化后系统服役寿命综合提升3.27倍。

关键概念

共振疲劳断裂（Resonance Fatigue Fracture）：当外部激励频率与结构固有频率一致时，振动幅值急剧放大，在应力集中区域（弹条跟部）产生循环疲劳损伤，最终导致断裂，是弹条失效的核心机制

刚度老化（Stiffness Aging）：底板垫板在轮轨疲劳载荷长期作用下，聚合物材料逐渐固化、分子间交联键密度增加，宏观表现为静刚度随服役时间线性升高，偏离设计工作点

磁流变弹性体（Magnetorheological Elastomer, MRE）：由聚合物基体与微米级软磁颗粒复合而成的智能材料，外加磁场时内部颗粒形成链状结构使刚度与阻尼可调，撤去磁场后可逆恢复，适用于实时补偿轨道垫板刚度退化

磁流变效应（Magnetorheological Effect）：MRE在磁场作用下刚度和阻尼发生可逆变化的物理效应，其微观机理为磁性颗粒链的形成与解体改变了材料的有效弹性模量

串联系统可靠性（Series System Reliability）：扣件系统各部件串联连接，任一部件失效即导致系统失效；系统总可靠性为各部件可靠性之积，且始终不高于最薄弱部件的可靠性

Goodman修正模型（Goodman Correction Model）：考虑平均应力对疲劳寿命影响的应力幅值修正方法，将非对称循环应力等效为对称循环应力，使S-N曲线可适用于实际服役工况

S-N曲线（S-N Curve）：描述材料在反复循环载荷下应力幅值（S）与疲劳寿命（N）关系的曲线，取对数后呈线性关系，本文拟合结果为log N = 11.43 − 1.75 log S（R²=0.959）

玻璃化转变温度（Glass Transition Temperature）：弹性高分子材料从高弹态转变为玻璃态的临界温度，对应-40°C，材料在此温度下变硬变脆，导致底板垫板可靠性最低

高频弹条（High-frequency Clip）：以远离轮轨磨耗激励频段为目标，通过调整弹条长度、高度等结构参数使固有频率避开共振区间的优化弹条，一二阶固有频率各提升约40%

主要贡献

  1. 系统揭示W300扣件主要部件失效机理：通过大规模现场调查（覆盖京沪、武广高铁，统计240块垫板及弹条断裂数据）与实验室验证，定量确认弹条共振疲劳断裂（高频激励占失效率98.1%）和底板垫板刚度线性老化两类失效模式的物理机制

  2. 建立多因素耦合寿命预测模型：为弹条建立同时考虑激励频率f和轮轨磨耗幅值λ的修正疲劳寿命模型；为底板垫板建立温度T与疲劳循环次数N耦合的刚度演变模型（K_T&N公式），为工程维修决策提供定量依据

  3. 开发高频弹条：仅通过结构参数调整（保持38Si7材料不变），使弹条一二阶固有频率提升约40%，疲劳寿命提升31.9%，同时具备降低成本的潜力

  4. 提出并验证MRE智能底板垫板：首次将磁流变弹性体应用于高铁底板垫板，实现刚度随磁场线性可调（最大提升47.8%），可模拟8～9年现场服役老化特性并实时补偿，为扣件系统全生命周期管理提供新途径

  5. 建立高铁扣件系统可靠性综合评估框架：基于串联系统可靠性理论，将激励频率、磨耗幅值、环境温度纳入统一模型，量化论证优化后系统综合寿命提升3.27倍，为新一代高铁扣件系统设计提供理论支撑

局限与展望

论文在以下方面存在一定局限性，可作为后续研究方向：

  • 数据共享受限：作者明确声明无权共享研究数据，可能影响模型的独立验证与推广应用
  • MRE工程化集成：论文验证了MRE垫板的材料性能和静动态特性，但对磁场施加装置的工程集成方案、长期运营中磁场设备可靠性及维护成本等问题未作深入讨论
  • 可靠性模型简化：串联模型假设弹条和底板垫板寿命均服从正态分布且标准差统一取均值的10%，实际分散性可能因线路条件和制造差异而有所不同；螺栓、轨距挡板等其他部件暂未纳入建模
  • 环境耦合效应：底板垫板刚度模型在温度-循环次数框架下建立，紫外线、湿度、化学介质等其他环境因素的耦合影响有待进一步量化
  • 长期现场验证：优化后新一代扣件系统在实际线路上的长期服役数据尚待积累，实验室预测寿命与现场寿命的吻合程度有待跟踪验证

相关工作

本文在综述部分明确讨论了以下既有工作：

  • Anat等：基于振动疲劳理论研究e型弹条的载荷与寿命关系，发现高载荷下疲劳寿命显著降低，奠定了弹条振动疲劳研究的基础框架

  • Xiao等：研究e型弹条模态、应力与磨耗激励频率的关系，发现弹条模态与轮轨磨耗频率一致会导致共振失效，与本文弹条失效机理相互印证

  • Wang等：研究ω型弹条振动传递函数，发现轮轨激励频段（520～630 Hz）与ω型弹条固有频率一致引发共振损伤，直接指向本文研究对象的共振失效问题

  • Xin等：通过车轨耦合动力分析模型研究弹条安装量程对疲劳寿命的影响，指出DI型弹条安装量程应控制在7～9 mm

  • Ma等：建立三维车轨耦合随机动力学模型，采用累积损伤理论和S-N曲线研究随机轨道不平顺对弹条疲劳损伤的影响，与本文S-N曲线建模方法一脉相承

  • Zhang等：研究服役8年的60Si2Mn弹条性能退化，发现防腐涂层脱落和腐蚀为主因，提示腐蚀这一失效途径与本文共振失效并存

  • Lu等：基于振动控制原理优化减振器，通过优化环形阻尼器避开高频振动共振，代表了改善弹条疲劳特性的另一技术路线

  • Gao等：基于ω型弹条共振失效机理改进结构参数和复合材料，成功规避轮轨磨耗激励共振，为本文弹条结构优化思路的直接前驱工作

  • Sung等：设计多种工况模拟底板垫板老化与刚度变化，对比新垫板热循环加载试验与在役垫板结果，获得热加速因子与循环载荷次数关系，为本文温度-循环次数耦合建模提供方法参考

  • Li等：现场调研发现CPU底板垫板性能衰减主要由高温老化引起，而非紫外线、臭氧或潮湿环境，支持本文将温度作为垫板老化核心影响因素的判断

  • Cui等：首次采用量子化学方法制备接枝受阻酚类抗氧剂的聚氨酯弹性体轨道垫板，热氧老化200天后仍能保持稳定静刚度，为提升橡胶类垫板耐老化性能提供了化学改性思路
