A novel algorithm for longitudinal track-bridge interactions considering loading history and using a new model for fastener behavior

2019-01-01

摘要

本文针对高速铁路桥上连续焊接轨的纵向轨-桥相互作用（TBI）问题，提出了一套考虑加载历史的新型分析算法。研究首先对WJ-8型扣件进行了系统的纵向力学试验，基于Dahl摩擦模型建立了考虑竖向荷载影响的扣件纵向阻力-位移曲线（LRDC）新变化模式，拟合精度优于既有的Ruge模型和双弹簧模型。在此基础上，利用Ritz方法和最小势能原理，推导了能够追踪荷载历史路径的TBI加载历史法（LHM）矩阵方程体系，涵盖季节性温度变化、列车竖向弯曲和制动三个典型荷载工况的序贯分析。将LHM与各规范推荐的线性叠加法（LSM）进行对比，以五跨简支梁桥为案例，从轨道额外应力、轨-桥相对位移（RRD）和墩顶纵向位移三项指标评估两种方法的差异。结果表明，LHM通常给出较小的轨道压应力，但得到更大的制动RRD和墩顶位移，两种方法的响应差异在很大程度上取决于所采用的扣件力学参数。

研究背景

连续焊接轨（CWR）因可减少轨缝、降低振动噪声、提高行车稳定性而被广泛铺设于高速铁路桥梁上。桥梁因温度变化或列车荷载产生结构变形时，会在轨道中诱发纵向附加力并引起轨-桥相对位移，即轨-桥相互作用（TBI）现象，对轨道结构安全和桥梁墩台受力均有重要影响。

TBI研究始于1974年Fryba提出基于线性纵向阻力的准静态拉普拉斯变换方法，此后经历了从理想弹塑性模型、DS899/59特殊规程到UIC 774-3R有限元推荐方法的演进，分析模型也从二维平面逐步发展至精细三维形式。目前，UIC、欧洲规范EN 1991-2、德国规范DIN Fachbericht 101及中国规范均推荐采用线性叠加法（LSM），即对各荷载工况独立计算后直接叠加。然而，LSM违背了问题本身的非线性特征，忽略了前序荷载事件对当前加载过程的影响，在某些情况下会导致对轨道承载力的显著高估或低估。

近年来，Ruge等人假设列车上桥时扣件阻力发生突变，开发了基于FORTRAN程序的非线性TBI模型；Zhang等人提出含卸载弹簧和附加弹簧的双弹簧模型，并以ABAQUS生死单元法实现。但两种模型在描述卸载到加载工况的LRDC变化规律方面均存在不足，前者的突变假设已被后续研究证伪，后者则系统性低估了扣件阻力。这一背景促使本文开展更准确的扣件本构建模和TBI分析算法研究。

核心内容

扣件力学试验与Dahl模型建立

对WJ-8型扣件进行系统试验，试验设备含竖向和纵向双向加载系统，T型螺栓扭矩保持设计值160 N·m，竖向荷载范围0至50 kN（步长10 kN），纵向力加载速率10 kN/min，加载至塑性滑移发生。

试验结果表明，随竖向荷载由0增至50 kN，扣件极限阻力Fult从15.03 kN增至45.62 kN，初始刚度k从6.139增至7.976 × 10⁶ kN/mm，临界弹性相对变形从2.45 mm增至5.72 mm，阻力比η从1.00增至3.04。空载工况拟合参数与中国规范值接近，但有载工况参数与各国规范差异显著，表明规范参数的代表性随荷载条件变化而变化。

基于Dahl摩擦模型（取参数α = 1.0得解析解），LRDC表达为：

F(x) = Fult · [1 − exp(−k·x / Fult)]

该公式由极限阻力Fult和初始刚度k两个具有明确力学含义的参数确定。

考虑加载历史的扣件新模型

当钢轨在位移x₀处施加竖向荷载时，扣件LRDC发生转换。论文通过在位移突变点x₀处以当前阻力值Fᵘ(x₀)为参考状态，对位移增大和减小两种情形分别建立描述曲线过渡过程的解析表达式，形成新的分段LRDC模型。

对比验证试验（在3 mm和10 mm位移处突加/突卸20 kN和30 kN竖向荷载）表明：Ruge模型的突变过程仅在加载转卸载情形下存在，在卸载转加载情形下不适用；双弹簧模型虽保证了连续性，但预测值系统性低于实测值；本文新模型在两种方向上均与实测结果吻合最优。

TBI-LHM算法推导

将扣件等效为非线性弹簧，推导各荷载阶段扣件势能的统一解析表达式，引入Heaviside阶跃函数和符号函数统一描述弹性区与塑性区。对于滞回特性下的新卸载曲线生成条件：当附加位移方向与前一加载步在滑动区的方向相反时，依据滞回特性生成新卸载曲线；否则继续沿初始卸载曲线演化。

在全局坐标系下建立包含N跨简支桥梁、两侧路基、扣件和墩台弹性弹簧的分析模型，利用Ritz方法将钢轨纵向变形展开为余弦级数（满足两端边界条件），结合最小势能原理建立以桥墩位移和轨道展开系数为未知量的分块矩阵方程。由于受载轨与非受载轨之间扣件阻力存在差异，两轨分别独立展开。

算法按三个荷载阶段组织为6个迭代步骤：温度变化阶段（Step 1–2）、弯曲力阶段（Step 3–4）和制动力阶段（Step 5–6）。每个新阶段开始时将扣件状态矩阵初始化，通过逐步更新符号函数矩阵进行迭代，直至收敛。前序阶段的位移参数通过历史修正项P显式传递至后续矩阵方程，确保荷载路径的完整性。全部推导在MATLAB平台上实现。

数值结果比较

以五跨简支梁桥为分析对象，采用UIC规范（UC）、德国规范（GC）和试验参数（PC）三套扣件数据，对LHM和LSM的计算结果进行系统比较：

  • 轨道额外压应力方面，LHM通常使最大压应力减小4.12%至8.58%，但对PC参数下受载轨影响甚微。
  • 制动RRD方面，LHM始终大于LSM：PC参数下偏大6.72%至19.79%；UC参数下偏大最为显著，受载侧和非受载侧分别偏大86.23%和46.52%。差异集中在靠近路基的270 m滑动支座附近，因为前序荷载已耗尽该区域扣件的弹性变形能力，LSM未能反映这一状态，导致制动阶段该区域扣件已进入滑动阶段而无法提供额外阻力，最终轨-桥相对位移显著增大。随桥跨数增加，GC和UC参数下最大RRD差异进一步扩大，分别达到46.2%至79.5%和49.7%至88.9%。
  • 墩顶纵向位移方面，PC参数下两方法差异极小（0.07%至0.65%），UC参数下LHM导致墩顶位移显著增大（14.80%至42.88%）。
  • 总体规律：两种方法的响应差异程度在很大程度上取决于扣件力学特性，PC参数下差异明显小于GC或UC参数下的情况。

关键概念

轨-桥相互作用（TBI）：桥上连续焊接轨因桥梁变形（热变形或荷载变形）与钢轨之间产生相对位移，从而在轨道中形成纵向附加力并作用于桥梁结构的耦合力学现象。

连续焊接轨（CWR）：消除接缝的长段焊接钢轨，广泛用于高速铁路以降低振动噪声和轨道维护需求，但因约束端部位移而产生较大温度力。

纵向阻力-位移曲线（LRDC）：描述扣件纵向抗力随轨-桥相对位移变化关系的本构曲线，是TBI分析的核心输入参数，在加载和卸载状态下表现不同。

Dahl摩擦模型：最早的动态摩擦模型之一，可描述小幅激励下的滞回行为及摩擦力对位移和相对速度的依赖性；取α = 1.0时得到双参数解析解F(x) = Fult[1 − exp(−kx/Fult)]。

线性叠加法（LSM）：各国规范推荐的TBI计算方法，对温度、弯曲、制动等荷载工况分别独立计算后线性叠加，未能反映加载历史的非线性影响。

加载历史法（LHM）：本文提出的TBI分析方法，通过追踪荷载序列中扣件的滞回状态，显式考虑前序荷载对当前工况的影响，揭示LSM在某些情况下的高估或低估问题。

Ritz方法：将钢轨纵向变形函数展开为满足边界条件的余弦级数，结合最小势能原理将连续场问题转化为有限自由度的代数方程组。

滞回特性：扣件在反复加卸载过程中，阻力-位移曲线形成封闭滞回环的性质；这是TBI-LHM区别于传统方法的本质属性，导致简单势能叠加方法失效。

轨-桥相对位移（RRD）：钢轨与桥面板之间的纵向相对滑移量，是评价TBI安全性的重要指标，制动工况下限值为4 mm。

极限阻力比η：有载扣件与空载扣件极限阻力之比；UIC规范取1.50，德国规范取2.00，中国规范取1.55，而本文试验表明该比值随竖向荷载增大可达3.04。

主要贡献

  1. 提出并验证了新的扣件LRDC过渡模型：针对卸载转加载工况，以当前位移和阻力状态为参考，对位移增大和减小两种情形建立分段解析表达式，克服了Ruge模型突变假设在该方向上的失效和双弹簧模型系统低估的缺陷，与实测结果的吻合精度最优。

  2. 建立了基于Dahl模型的扣件参数体系：通过WJ-8扣件系统试验，揭示了竖向荷载对极限阻力、初始刚度和临界弹性变形的定量影响规律，发现有载工况参数与各国规范差异显著，为规范修订提供了数据依据。

  3. 推导了完整的TBI-LHM矩阵方程体系：结合Ritz展开、最小势能原理和统一势能解析表达式，将三阶段荷载序列（温度→弯曲→制动）的TBI问题归结为6步迭代算法，实现了对加载历史路径的完整追踪，弥补了规范LSM在非线性问题上的理论缺陷。

  4. 定量揭示了LSM的适用边界：通过系统对比研究，表明采用欧洲规范或德国规范扣件参数时，LHM与LSM的制动RRD差异可达46%至89%，而采用试验参数时差异相对较小，为工程实践中评估规范方法误差提供了量化参考。

局限与展望

方法局限：

  • 分析对象限于简支梁桥，桥跨数从1至5跨，未涵盖连续梁桥等其他结构形式。
  • 桥梁弯曲分析采用均布活载的静力近似，未考虑动力效应。
  • 制动力采用常数均布模型，实际列车制动荷载分布更为复杂。
  • 忽略了滑动支座摩擦力的影响。

未来研究方向：

  • 将本文模型推广至大跨度连续梁桥，该结构中轨道与桥面板之间的相互作用更为强烈，非线性效应预期更显著。
  • 深入研究列车离桥后的钢轨上拱效应，以及桥梁降温工况下的残余变形与残余应力对扣件和钢轨长期服役性能的影响。
  • 考虑多次列车过桥循环作用下扣件累积损伤与TBI系统响应演化。

相关工作

本文正文中明确讨论的既有工作如下：

  • Fryba（1974）：提出基于线性纵向阻力的TBI准静态分析方法，采用拉普拉斯变换求解，但忽略桥梁挠度，大跨度情况下精度下降；为TBI定量分析的早期奠基性工作。

  • Arya和Agrawal（1982）：采用理想弹塑性纵向阻力模型，将墩顶位移和桥梁挠度纳入分析；但方法需反复假设和修正路基上的轨道长度，计算效率受限。

  • 德国铁路桥梁局（1985，DS899/59）：发布新干线铁路桥梁特殊规程，首次在加载和卸载工况下引入不同的理想弹塑性纵向阻力模型，提出轨-桥连接可用拉压杆或弯梁模拟。

  • UIC 774-3R（1995/2001）：综合欧洲工程经验建立桥上CWR验算标准，推荐采用有限元方法计算附加纵向轨道力，成为欧洲TBI分析的主要依据。

  • Ruge等人：假设列车上桥时扣件阻力发生突变，基于FORTRAN程序开发了非线性TBI模型；本文指出其突变假设在卸载转加载方向上不适用，且摩擦理论表明竖向列车荷载应强化轨-桥连接而非触发突变。

  • Zhang等人：提出含卸载工况弹簧和附加弹簧的双弹簧模型，利用ABAQUS生死单元法实现，并应用于城市轨道交通七跨简支U型梁桥；本文试验验证表明该模型预测值系统性低于实测数据。
