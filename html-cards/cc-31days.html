TACTICS

Key Features

自动初始化

拒绝手写文档。输入 /init 让 Claude 主动阅读代码库，自动生成 CLAUDE.md。它不仅是说明书，更是 AI 的项目记忆，涵盖构建命令与架构决策。

瞬时终端

无需切换窗口。使用 ! 前缀直接在对话流中执行 Bash 命令（如 !git status）。输出即刻注入上下文，实现零延迟的无缝交互。

时光回溯

实验性的代码修改搞砸了？双击 Esc 键即可将对话与代码状态回滚至安全检查点。保留探索的勇气，同时拥有随时撤销的后悔药。

语境注入

善用 @ 符号实现毫秒级上下文引用。无论是 @src 还是 @mcp，这是将人类意图与 AI 认知同步的最快路径，支持智能模糊匹配。

思维链涌现

面对复杂架构，通过 ultrathink 强制调用 32k 思考 token。在输出代码前进行深度推理与自我博弈，用计算成本换取工程洞察。

会话持久化

无论是断电还是跨设备，--resume 让你随时找回工作流。为会话命名，像管理 Git 分支一样管理你的 AI 协作历史。

安全沙箱

通过 /sandbox 预设权限边界。在 YOLO 模式的极致速度与严格授权之间找到平衡，确保 AI 在安全的笼中起舞。

无头自动化

并非所有任务都需要对话。使用 -p 启动 Headless 模式，将 Claude 接入 CI/CD 流水线，处理 Lint 修复并直接输出结果。

智能体群

复杂任务不必亲力亲为。调用 Subagents 处理并行子任务，利用 MCP 连接外部工具。从单兵作战进化为指挥官模式。
