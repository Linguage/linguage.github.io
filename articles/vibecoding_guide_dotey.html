规划流程

先写详尽 Markdown 计划 → 评审精简 → 小步快跑 → 标记“暂不开发”，防范围蔓延。

版本控制

Git 是生命线；每功能前干净 commit，AI 一旦“天马行空”立即 reset --hard，拒绝补丁堆积。

测试护栏

优先端到端集成测试，模拟真实用户点击；先写测试再给 AI 边界，回归问题立即现形。

高效修 Bug

完整错误日志直接贴 AI → 分析根因 → 失败就 reset → 加日志/换模型 → 干净修复。

工具优化

cursor.rules / windsurf.rules 写清规范；本地放 API 文档；前端 Cursor、后端 Windsurf 各取所长。

复杂功能

独立原型库先跑通 → 保持 API 一致 → 模块化拆分；拒绝万行神文件，monorepo 先靠边。

“AI 能 1 小时干完你 1 周的活，也能悄悄埋雷——用工程化流程把它变成靠谱合伙人。” Code, Test, Reset, Refactor — Repeat.

立即行动

Write plan → Git init → Test first → Prompt in chunks → Reset if stinks → Ship.
