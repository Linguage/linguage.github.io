Vibe Coding 定义

用自然语言驱动 LLM，将开发流程与 prompt 工程高度融合，像“新型编程语言”一样与 AI 协作。

分段式推进

先共创详尽计划 Markdown，再逐模块实现、测试、提交；拒绝“一步登天”。

测试先行

人工手写测试用例 → 让 LLM 生成代码；用集成测试兜住真实链路，防回归。

Git 快照回滚

每步功能前 commit；LLM 陷入死循环立即 reset，拒绝“补丁碎片”堆积。

工具混搭

前端/后端分别跑 Cursor vs Windsurf，对比选优；定期换最新模型，Gemini 规划、Sonet 3.7 实现。

持续重构

功能通过后主动请 LLM 点评代码，小文件、高内聚，保持人机可读。

“AI 尚不能单步产出复杂产品，但工程化节点 + 复盘迭代，让它成为靠谱合伙人。” Engineering + Feedback Loops = AI Co-Pilot

立即行动

Clone a side project → Write tests → Prompt in segments → Git reset when stuck → Ship & refactor.
