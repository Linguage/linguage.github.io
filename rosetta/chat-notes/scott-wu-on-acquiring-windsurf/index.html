访谈介绍

本访谈录记录了20VC主持人Harry Stebbings与Cognition公司CEO Scott Wu的深度对话。访谈的核心围绕Cognition近期完成的重磅交易——在72小时内成功收购Windsurf公司展开。Scott Wu详细回顾了这次快速收购的决策过程、背后的商业逻辑，以及双方在团队、产品与市场方面的互补性。

对话并未止步于单一的商业事件。基于此次收购，访谈进一步延伸至对当前AI行业的宏观洞察。内容涵盖了AI领域激烈的人才竞争、价值在产业链各环节的分布与归属、基础模型与上层应用之间的竞合关系等核心议题。Scott Wu分享了他对AI技术发展趋势的看法，特别是对强化学习（RL）潜力的强调，以及他对行业估值泡沫论的反驳。

此外，访谈还深入探讨了AI对软件工程未来的颠覆性影响，包括AI代理（Agents）如何重塑开发流程、工程师的角色将如何演变，以及Cognition对下一代人机交互界面的构想。

一：Cognition 72小时闪电收购Windsurf：AI时代的速度与远见

引言 Scott Wu，Cognition的联合创始人兼CEO，在近期的一场访谈中，详细披露了其公司如何在短短72小时内完成对Windsurf的年度收购案。这宗交易不仅是AI领域的一大新闻，更折射出当前技术变革时代下，企业决策、价值判断与执行速度的新范式。本文将深入解析这场“闪电战”的幕后逻辑，以及它对AI行业并购所带来的启示。

核心要点一：机会的捕捉与价值的重估

  • 突如其来的契机： Scott Wu透露，Cognition与外界几乎同时在周五得知Windsurf的潜在拆分信息。迅速识别出Windsurf作为“宝藏”而非“空壳”的潜力，是Cognition能够迅速行动的关键。
  • Google是否错失良机： 访谈中提出Google是否“错失了金矿”的问题。Scott Wu的回答暗示，大型企业在组织重构时，确实可能忽略某些“被遗留”的资产的真实价值。他强调，Windsurf拥有出色的产品、完整的客户群、全部代码、专有IP以及一支强大的团队，这些都是其被Cognition看重的核心价值。
  • 互补性是核心驱动： Cognition专注于核心工程与产品团队，而Windsurf则在市场推广、营销、财务和运营方面具备优势。这种天然的互补性，使得Scott Wu认为这是一次“非常自然的合作”，极大地加速了决策过程。

核心要点二：AI时代下的极速并购逻辑

  • “周五晚银行托管”的紧迫感： Scott Wu将此次收购的急迫性比作“银行在周五晚进入托管，必须在周一给出答案”。他强调，在这种行业巨变且信息高度透明的环境中，客户、团队和整个硅谷都在观望。快速行动是为了稳定军心，向市场传递明确信号，避免价值在观望和等待中流失。
  • 牺牲深度尽调换取速度： Cognition承认，72小时内无法进行如常规M&A那般深入的尽职调查。但凭借对AI领域业务的深刻理解和对Windsurf高层（如Jeff、Graham和Kevin）的信任，他们认为在高层次上已能充分理解业务。这表明在特定背景下，战略契合与速度优先于极致的细节核查。
  • 谈判与执行的效率： 从周五晚的首次接触，到周六口头协议，周日敲定所有法律和商业细节，最终在周一上午完成签署并宣布，整个过程展现出惊人的效率和协调能力。Scott Wu特别赞扬了Windsurf团队为此付出的“不眠不休”的努力，这凸显了交易双方在紧迫关头下的高水平协同。

核心要点三：创始人与并购的“新常态”

  • “与船共沉”契约的演变： Scott Wu提到，传统的“创始人与船共沉”的契约在过去一年有所改变，他对此感到“有点失望”。这暗示了在快速变化且充满不确定性的AI时代，创始人和团队面对的退出或整合路径变得更加多样，而不再是简单的“成功”或“失败”。
  • 对未来并购模式的启示： 尽管Scott Wu认为现在断言这种极速收购会成为“新常态”还为时过早，但Windsurf案例无疑提供了一个新的模板。它表明，当战略契合度极高、市场窗口期极短时，速度、互信和对核心价值的精准判断将成为决定交易成败的关键因素。

结论 Cognition对Windsurf的闪电收购，不仅仅是一次简单的商业交易，更是AI时代下企业如何捕捉机遇、评估价值、快速决策并高效执行的生动案例。它提醒我们，在技术飞速迭代的今天，市场竞争的节奏已然加快，敏锐的洞察、果敢的行动以及对核心人才与资产的精准把握，将成为企业制胜未来的重要筹码。

二：重新定义软件工程师：AI Agents如何重塑软件工程的未来

引言 在人工智能浪潮席卷全球的当下，软件工程领域正经历着前所未有的变革。Cognition CEO Scott Wu在近期访谈中深入探讨了AI，特别是AI Agents，将如何根本性地重塑工程师的角色与工作方式。他大胆预测，未来软件工程师的效率将实现“10倍”提升，而我们所理解的“代码编写”也将被“意图表达”所取代。

核心要点一：效率的指数级跃升与代码的“生产力悖论”

  • 当前效率已显著提升： Scott Wu指出，目前使用AI工具的工程师效率已达到未使用的工程师的 $1.5$ 到 $2$ 倍。他引用Salesforce和Robinhood高管的观点，即高达 $50\%$ 的新代码已由AI生成，尽管他强调需考虑代码的重要性与AI辅助的程度。
  • 未来 $10$ 倍效率的展望： Scott Wu乐观预测，在未来三年内，工程师效率将达到 $10$ 倍提升。更令人兴奋的是，他认为这种效率的提升将引发“Jevons悖论”——即效率的提高将导致代码总量的 $10$ 倍增长。
  • “代码饥渴症”： Scott Wu生动地描述了当前软件的普遍缺陷，从银行APP到医疗系统，再到保险服务，许多软件的用户体验远未达到最佳。他认为，市面上所有软件都可提升 $10$ 倍，且有 $10$ 倍多的新软件需要被编写。AI的赋能将释放巨大的创造力，满足这种未被满足的“代码饥渴”。

核心要点二：Agent的终极愿景：从“编写”到“拥有”与“表达”

  • Agent接管“所有权”： Scott Wu的核心观点是，未来的AI Agent将不仅仅是代码助手，而是真正能够“接管工作的所有权”。这意味着Agent将能独立完成从理解需求到执行、调试和部署的整个软件开发生命周期。
  • “Tony Stark与Jarvis”： 他用Tony Stark与Jarvis的交互来描绘未来的理想场景——工程师不再需要打开笔记本电脑编写代码，而是直接通过表达意图来指挥计算机完成复杂任务。代码将从“人机交流的语言”退化为“计算机内部执行的细节”。
  • 角色转型：技术架构师与产品经理： 在Agent深度参与的未来，工程师的价值将不再体现在编写代码的能力，而是聚焦于更高级别的“决策”。他们将更多地扮演技术架构师、技术产品经理的角色，专注于定义问题、构思解决方案、设计系统架构，以及与Agent进行高层次的沟通与协作。
  • 深层语境是当前挑战： Scott Wu指出，目前Agent面临的最大挑战是“深层语境”的理解——即在大型复杂代码库中建立直觉、理解各组件交互，并有效利用各种工具进行调试和诊断。这不是沙盒问题，而是高度实践性的挑战。

核心要点三：IDE与Agent的融合：人机协作的新范式

  • 互补的过渡阶段： Scott Wu认为，在AI Agent完全成熟之前，IDE（集成开发环境）和Agent将在未来几年内成为开发者工作流中不可或缺的一部分。他预见，IDE与Agent的结合将创造全新的、无缝的“同步到异步再到同步”的工作体验。
  • 理想的工作流程： 工程师可以在IDE中规划任务，利用AI的智能检索和上下文理解来制定决策；然后将大部分繁重的工作交给Agent异步执行；最后，在IDE中审查代码，并进行必要的本地修改。这种模式将最大化人类与AI的协作效率，让工程师能够专注于创造性与战略性任务。
  • Devin与Windsurf的融合愿景： 收购Windsurf后，Cognition将致力于探索如何将Devin（AI Agent）与Windsurf（IDE）的优势结合。目标是为同时使用两者的用户提供极致流畅的体验，实现人机协同的最高境界。

结论 Scott Wu对AI Agents和软件工程未来的展望是激动人心的。它预示着一个工程师将从繁琐的编码工作中解放出来，转而专注于更高价值的创新与架构的时代。这不仅是对效率的简单提升，更是对软件工程师职业定义，乃至人机交互方式的深远重塑。我们正站在一个新时代的门槛上，见证着人类智能与人工智能的边界在代码世界中不断消融。

三：AI的宏观脉动：人才争夺、价值流向与强化学习的隐形引擎

引言 当前，人工智能领域正经历着爆炸式增长，引发了关于人才、价值创造和技术突破的广泛讨论。Cognition CEO Scott Wu在一次深度访谈中，分享了他对AI宏观格局的独特见解，包括对人才战的看法、AI价值在产业链中的分布，以及被他视为“隐形引擎”的强化学习（RL）的深远影响。

核心要点一：人才争夺的本质与AI核心群体

  • “合理”的人才战： 针对外界普遍认为AI人才争夺“失控”的观点，Scott Wu提出了一个“也许疯狂、但合理”的看法。他认为，鉴于AI是“我们一生中最大的技术变革”，对顶尖人才的投入是完全合理的，甚至仍显不足。
  • 塑造AI未来的“极少数人”： 他指出，真正决定AI发展轨迹的人非常稀少，数量大约在 $100$ 到 $1000$ 人之间，肯定远少于 $10000$ 人。这个精英群体是全球AI进步的核心驱动力，他们的稀缺性直接导致了激烈的人才竞争。
  • 应用层人才竞争的激烈： 令人惊讶的是，Scott Wu认为，即使在应用层，人才竞争的激烈程度也可能不亚于基础模型实验室。这表明AI的价值创造正向更广阔的领域渗透，对具备AI应用能力的综合型人才需求旺盛。

核心要点二：AI价值的多元化归属与市场格局

  • 价值在于“差异化”： Scott Wu驳斥了“AI价值只归属于芯片、基础模型或应用层”的单一论调。他强调，价值将累积在任何能够建立“真正差异化”的领域。这意味着无论是在基础设施、基础模型还是应用层面，只要能提供独特且不可替代的价值，就能在AI生态中占据一席之地。
  • 应用层与基础模型层的共生关系： Scott Wu认为，应用层（如Cognition）与基础模型层（如Anthropic）的关系更多是“合作”而非简单的“竞争”。他指出，双方关注的问题不同：基础模型旨在提升通用智能，而应用层则专注于如何将这种智能优化并应用于特定垂直领域（如软件工程）的具体产品体验。
  • 驳斥“泡沫论”与投资前景： 面对普遍存在的“AI泡沫”论调，Scott Wu表现出极强的信心。他引用Sam Altman十年前对YC公司估值的“泡沫论”反驳，认为AI的价值创造潜力被严重低估。他坚信，无论基础模型公司（如OpenAI、Anthropic）还是应用层公司（如Perplexity、Cognition），其整体估值在未来五年都将大幅增长，两者都是“绝佳的投资”。
  • 基础模型层的整合趋势： 尽管市场潜力巨大，Scott Wu预测基础模型层最终将走向整合，全球可能只剩下 $2$ 到 $6$ 家（更可能 $3$ 到 $5$ 家）主要玩家。这种整合将是市场效率和资源集中的自然结果。

核心要点三：被低估的“隐形引擎”——强化学习（RL）

  • RL：过去一年半的最大突破： Scott Wu明确指出，强化学习（RL）是过去一年半AI能力上“最大的突破”，其潜力被人们严重低估。他认为，RL使得模型能够“解决任何基准问题”，只要有清晰的成功/失败反馈和明确的环境定义。
  • 从“模仿学习”到“行为优化”： 他解释道，早期的模型（如GPT-3/3.5）主要基于“模仿学习”，即通过阅读海量互联网数据来模仿人类行为。而RL则更进一步，通过不断试错和奖励机制来优化特定行为，使其在特定任务上超越模仿。
  • 数据范式的转变：质量重于数量： RL的崛起也带来了数据范式的深刻转变。Scott Wu观察到，AI领域正从追求“海量数据”转向追求“一小部分高度策展的、针对特定用例的”高质量数据。他以Cognition的Kevin-32B模型为例，证明了通过RL在少量高度特定数据上的训练，能让模型在特定任务（如CUDA kernels编写）上取得远超传统SOTA模型的表现。这意味着未来AI的突破将更多依赖于“计算”和“高质量数据”，而非单纯的“大数据”。

结论 Scott Wu的访谈勾勒出AI时代下一幅充满活力与挑战的画卷。他不仅为我们揭示了AI浪潮下企业并购的全新速度与逻辑，更通过对人才、价值和核心技术（特别是强化学习）的深度剖析，展现了AI行业远未被完全理解的巨大潜能。这场变革不仅仅是技术的革新，更是商业模式、人才结构乃至人类工作与生活方式的全面重塑。

Cognition 的增长与营销

  • Devin 的增长： 在过去 $6$ 个月（1 月至今），Devin 的使用量增长了约 $5$ 到 $10$ 倍，包括自助服务和企业客户。
  • 收入增长： 即使不包括 Windsurf 的交易，Cognition 在过去 $6$ 到 $8$ 个月内也实现了从约 $0$ 到 $8000$ 万美元的年化经常性收入（ARR）增长。
  • 营销策略： Devin 的增长主要来自真实的工程团队在 Slack 和 Linear 中标记 Devin，并在 GitHub 中使用其拉取请求，它专注于为工程师提供加速工作流程的工具，而不是面向普通消费者。
  • 未来的营销： Scott Wu 承认 Cognition 在销售和营销方面可以做得更好，现在收购了 Windsurf 的营销团队，希望有所改善。
  • 市场格局： Scott Wu 认为现在谈论市场格局还为时过早，因为软件工程的未来远比现在设想的要先进。

IDEs 与 Agents：未来的训练轮？

  • 人机界面的演进： IDEs 和编码 Agent 是下一代人机界面的开端。最终，用户将通过表达意图来命令计算机，而不是通过代码。
  • Tony Stark 的例子： Scott Wu 举例 Tony Stark 不需要打开笔记本电脑，而是与 Jarvis 对话来完成工作，预示了未来直观的人机交互。
  • 未来展望： 这一演变需要几年时间，但每隔 $2$ 到 $3$ 个月就会有新的产品体验进步。

竞争与独特方法

  • 竞争对手： Scott Wu 认为 NVIDIA 的 Jensen Huang 曾说过，“当你找到公司获胜的方式，意味着没有人需要失败”，这表明每个公司都有其独特的利基市场。
  • Cognition 的独特性： Cognition 始终拥有独特的方法和对未来愿景的看法，这使其有别于其他公司。
  • 多个赢家： Scott Wu 坚信 AI 编码领域会有多个赢家。

未建造的渴望与整合体验

  • 整合 IDE 和 Agent 体验： Cognition 最想构建的是 IDE 和 Agent 的结合体验。
  • 未来工作流： 开发者可以在 IDE 中规划任务，利用 AI 的智能和信息检索功能，然后将大部分工作交给 Agent 完成，最后在 IDE 中审核和修改代码。
  • 同步与异步： 这种结合将实现同步与异步工作的无缝切换，让开发者在需要时介入，并在不需要时让 Agent 并行处理。
  • Windsurf 和 Devin 的结合： 在短期内，将保持 Windsurf 和 Devin 产品的各自理念，但会探索如何为同时使用两者的用户提供平滑的切换体验。

快问快答

  • 睡眠情况： Scott Wu 承认过去一周因交易和团队整合而睡眠不足。
  • 一个被广泛持有的 AI 错误信念： 人们低估了强化学习（RL）的力量和潜力。
    • RL 的进步： 过去一年半，RL 是 AI 能力上的最大突破。它能解决几乎所有基准问题，只要能清晰定义所需的行为、操作环境以及成功或失败的条件。
    • 数据的重要性转变： AI 领域在 $24$ 个月前关注的是海量数据，现在则更多关注一小部分高度策展的、针对特定用例的数据。例如，通过 RL 在 CUDA kernel 代理轨迹上的训练，可以显著提高模型在相关任务上的表现。
    • 结论： 更多的是计算（compute），而不是海量数据；数据质量重于数量。
  • 对新学生学习计算机科学的建议： 绝对应该学习 CS。CS 不仅仅是学习具体的技术，更是学习如何从第一性原理思考问题，以及理解计算机模型和架构（如数据库工作原理）。这些是永远有价值的技能。
  • 最引以为傲的个人特质： 情绪上的冷静。尽管他本人好胜且容易沮丧，但在压力情境下（尤其是在公司和近期交易中）能够保持镇定。
  • 营销建议： 像 Nike 一样，让用户感觉自己是“超级英雄”，通过人类增强的故事来营销。同时，更直接地分享 Cognition 自己的成长故事，主动塑造叙事，避免流言蜚语。Scott Wu 承认，过去公司更倾向于低调，但现在大家都知道 Agent 技术在发展，应该更加公开。
