摘要

本文提出了非椭圆指数和共形指数作为定量评估指标，用于评估非赫兹轮轨接触对多体模拟框架内车辆动力学的影响。评估使用在线和离线方法进行，采用Simpack和CONTACT软件，涉及单个轮对-轨道系统和完整车辆-轨道模型在各种运行条件下的比较。数值模拟结果表明，接触斑的非椭圆性和共形性都会导致接触力偏差，并且它们的影响是耦合的。在线模式下观察到的偏差比离线模式下小，主要是由于Simpack和CONTACT中应用的等效穿透不同。所提出的指数为评估非赫兹接触的重要性提供了统计基础，为确定这些影响是否可以在动态模拟中被忽略提供了实用手段。

关键词

#非椭圆接触 #共形接触 #多体模拟 #接触指数 #车辆动力学

背景与动机

传统上，赫兹理论被广泛用于建模轮轨接触，特别是在多体系统（MBS）模拟中，由于其高计算效率。然而，赫兹接触模型在准确表示真实轮轨接触条件方面存在局限性，这主要是由于轮轨剖面的高度非线性特性。这些限制主要源于其假设椭圆接触形状和平坦接触面积。

为解决这个问题，开发了非赫兹接触方法。本文中广泛使用非赫兹术语，包括非椭圆和共形接触条件。非椭圆模型旨在考虑偏离椭圆的接触形状，同时仍然假设平坦接触斑。相比之下，共形模型考虑接触斑弯曲的情况。

研究已经证明了赫兹（平坦椭圆接触面积）与非椭圆或共形接触解决方案之间的偏差。然而，这些差异取决于具体场景。这就提出了一个关键问题：在什么条件下多体模拟需要非赫兹接触模型？本研究通过系统方法初步回答这个问题。

研究方法

本研究提出了一种系统的非赫兹接触影响评估框架：

  1. 定量评估指数：

    • 非椭圆指数：I_ne = (y_l + y_r) / (y_r - y_l)，其中y_l和y_r是接触斑左右边界的横向坐标
    • 共形指数：I_cf = (δ_l - δ_r) / (π/2)，其中δ_l和δ_r是接触斑左右边界的接触角
    • 两个指数值范围均为0到1，值越大表示非赫兹特性越明显

  2. 评估方法体系：

    • 在线评估：使用Simpack软件的EEC和DEC方法进行两次模拟，比较结果
    • 离线评估：先使用EEC方法生成系统状态时间历程，然后将状态输入CONTACT进行比较
    • 软件工具：Simpack（MBS软件）和CONTACT（高级接触建模工具）

  3. 接触模型比较：

    • EEC方法：等效弹性接触，将相互穿透区域转换为等效椭圆，使用FASTSIM计算蠕滑力
    • DEC方法：离散弹性接触，将接触区域分割成切片，在每个切片上应用FASTSIM算法
    • CONTACT方法：提供平面接触方法（考虑非椭圆性）和共形接触方法（考虑椭圆性和共形性）

  4. 测试场景设计：

    • 单个轮对-轨道系统：锥形轮/圆柱轨（赫兹条件）和S1002轮/UIC60轨（非赫兹条件）
    • 完整车辆-轨道系统：曼彻斯特基准客车模型，不同曲线半径和速度组合
    • 载荷条件：100kN垂直载荷，30kN横向载荷，考虑轨道不平顺

主要结论

  1. 指数有效性验证：

    • 所提出的非椭圆指数和共形指数能够有效量化非赫兹轮轨接触对车辆动力学的影响
    • 两个指数在接触力偏差中都起着重要且相互关联的作用
    • 为识别何时必须考虑非赫兹影响提供了实用和统计手段

  2. 在线评估结果：

    • 在赫兹条件下（Case 1），两个指数都接近零，EEC和DEC方法间差异很小
    • 在非赫兹条件下（Case 2），轮缘处出现共形接触，观察到显著更大的力偏差
    • 横向力比垂直力对非赫兹效应更敏感，共形性对横向力影响更大

  3. 离线评估结果：

    • 离线模式下的偏差明显大于在线模式，主要由于接触模型中穿透实现方式的差异
    • 共形性在离线评估中比非椭圆性具有更大影响
    • 20%的归一化垂直和横向力差异分别超过41%和39%，表明需要考虑非赫兹效应

  4. 统计阈值分析：

    • 当80%的非椭圆指数值低于0.13且80%的共形指数值低于0.42时，力偏差保持较小
    • 垂直和横向力的80%分位数分别为0.35%和2.55%，表明在此条件下可以忽略非赫兹效应
    • 指数影响呈现非线性特征，临界阈值约为0.1

  5. 模型选择建议：

    • 在线模式：垂直力偏差几乎与非椭圆指数成正比，横向力偏差呈现非线性行为
    • 离线模式：共形性比非椭圆性对观察到的变化具有更主导的影响
    • 临界条件：当非椭圆指数和共形指数超过特定阈值时，应使用非赫兹接触模型

  6. 局限性说明：

    • Simpack的DEC或CONTACT提供的近似方法都没有完全考虑共形性
    • 实际效应可能比观察到的更显著
    • 不同接触模型中使用的穿透方法存在差异，需要进一步研究

文章亮点或创新点

  1. 创新性指数：首次提出非椭圆指数和共形指数作为定量评估非赫兹接触影响的指标
  2. 系统评估框架：建立了完整的在线和离线评估体系，用于比较不同接触模型
  3. 统计基础：为确定何时可以忽略非赫兹效应提供了统计依据和实用方法
  4. 多场景验证：通过单个轮对和完整车辆模型在不同运行条件下验证了方法的有效性
  5. 工程指导价值：为多体模拟中接触模型的选择提供了明确的指导原则
