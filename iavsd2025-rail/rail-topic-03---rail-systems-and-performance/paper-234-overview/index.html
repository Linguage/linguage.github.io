  • 中文标题：浮置板轨道与准零刚度机构和动态减振器集成用于低频振动减缓
  • 作者及单位：Yuhao Ren, Shengyang Zhu*, Wanming Zhai (西南交通大学，中国)

摘要

钢弹簧浮置板轨道(FST)固有频率处的振动放大仍然是一个重大问题。本研究将准零刚度机制结合动态减振器(QZSDVA)集成到FST中，形成了一种新型的减振轨道(QZSDVA-based FST)，以有效抑制低频轨道振动。推导了二自由度QZSDVA系统动态响应的解析表达式，并通过数值方法进行了验证。在此基础上，建立了车辆-QZSDVA-based FST动态模型，并通过H∞优化指导确定了DVA的最优参数。进一步的探索强调了QZSDVA-based FST在显著抑制轨道振动方面的能力。结果表明，优化调谐的参数使QZSDVA-based FST能够有效减少低频振动，即使使用最小的DVA质量。

背景与动机

浮置板轨道(FST)系统因其能够减轻振动而被广泛实施，特别是在对这种振动敏感的区域。然而，尽管FST有效隔离了高频振动，但它在其固有频率（约10 Hz）附近放大了低频振动。受QZSDVA理论启发，本文通过将动态减振器和准零刚度支撑结构相结合，提出了一种新颖的低频振动衰减浮置板轨道(QZSDVA-based FST)。QZSDVA-based FST的创新来自于 blending DVA-FST和QZS-FST的优势。它通过DVA吸收振动能量，同时防止振动从轨道结构传递到周围环境。与DVA-FST相比，QZSDVA-based FST表现出显著的属性，能够有效抑制浮置板基础频率处的振动，而不会显著放大超低频振动或在其他频率处出现新的共振峰。此外，QZSDVA-based FST作为被动式减振轨道运行，消除了对外部能源的需求，从而抑制了潜在的施工和维护成本上涨。

研究方法

QZSDVA系统如图1所示，包括主系统、DVA和负刚度机制。后者包括一个通过水平弹簧连接到基础的滚轮，以及一个牢固地连接到主系统的半球。参数m1、k1和c1表示主系统的质量、刚度和阻尼系数，而m2、k2和c2表示DVA的相应参数。负刚度机制中的水平弹簧刚度用kh表示。半球和滚轮半径分别用r1和r2表示。假设系统最初处于静态平衡位置，半球和滚轮的中心处于相同高度。水平弹簧的原始长度为lh，预压缩值为P0。QZSDVA系统是一个具有两个自由度的非线性系统。本节将特别专注于系统在频域中稳态振动响应的解析确定。主系统受到谐波载荷F = F0 sin(ωt)的作用。引入U = Z1 - Z2作为主系统和DVA之间的相对位移，以及以下无量纲参数：无量纲激励力幅度f = F0/[k1(r1 + r2)]和无量纲相对位移u = U/(r1 + r2)，控制QZSDVA系统行为的无量纲动态微分方程为：

$$\ddot{z}_1 + 2\xi_1\omega_1\dot{z}_1 + f_v(z_1) + 2\mu\xi_2\omega_2\dot{u} + \mu\omega_2^2 u = f_0 \sin(\Omega t)$$

$$\ddot{u} + 2\xi_2\omega_2\dot{u} + \omega_2^2 u = \ddot{z}_1$$

主要结论

QZSDVA-based FST能够显著减少浮置板的峰值加速度，同时有效抑制6.3-16 Hz频率范围内轨道结构的低频振动。当地铁列车通过时，FST中浮置板的最大加速度为4.45 m/s²，而在QZSDVA-based FST中，它降至2.38 m/s²，减少了46.52%。在三分之一倍频程谱中，FST中浮置板的振动集中在6.3 Hz至16 Hz之间，最大振动发生在10 Hz，高达102.61 dB，这可能是由于浮置板的低频共振。与FST相比，QZSDVA-based FST在6.3-16 Hz频率范围内表现出显著的振动衰减效果。系统参数的最优设计使QZSDVA-based FST能够将10 Hz处的振动水平衰减19.42 dB，从102.61 dB降至83.19 dB。值得注意的是，QZSDVA-based FST可能在某些超低频范围内放大振动。例如，在4 Hz处，QZSDVA-based FST将振动水平从61.59 dB放大到78.05 dB，增加了16.46 dB。这种现象的可能原因可归因于负刚度机制的引入，这使浮置板轨道系统具有更大的灵活性并降低了其固有频率。然而，考虑到其在其他频带的出色振动衰减性能，这一缺点是可以接受的。

文章亮点或创新点

  1. 创新性集成设计：首次将准零刚度机制与动态减振器集成到浮置板轨道中，形成了一种新型的QZSDVA-based FST减振轨道系统
  2. 解析方法突破：推导了二自由度QZSDVA系统动态响应的解析表达式，为非线性振动系统的分析提供了新的理论工具
  3. H∞优化应用：采用H∞优化方法确定DVA的最优参数，实现了系统振动吸收能力的最优化
  4. 显著减振效果：实验证明该系统能够将浮置板加速度减少46.52%，在10 Hz关键频率处振动水平衰减19.42 dB，为低频振动控制提供了有效解决方案
