摘要

列车脱轨是严重碰撞中造成人员伤亡的主要因素，因此深入研究列车被动安全设计对减少碰撞事故损失具有重要意义。本文分析了车钩粘弹性阻尼器的特性，将其等效为变刚度弹簧和阻尼元件组成的系统。结合缝隙层流、薄孔流动及粘弹性材料的可压缩性特性，分别计算弹性力和阻尼力，建立相应的数学模型。通过与台车试验对比验证模型可靠性。其次，基于列车碰撞动力学理论，引入速度相关车钩模型进一步修正碰撞动力学方程。最后分析了头车车钩各参数对轮对最大抬升量的影响。研究结果为优化列车被动安全设计提供理论支持，对降低不同碰撞速度下的脱轨风险具有重要指导意义。

背景与动机

在碰撞事故中，列车脱轨是造成严重人员伤亡的主要因素之一，突显了优化机车车辆被动安全设计的重要性。作为一种高效低碳的公共交通方式，城市轨道交通已成为许多国家和地区不可或缺的基础设施。当前地铁车辆抗撞标准主要参照EN 15227:2020，围绕25km/h碰撞速度建立设计要求。然而在北美和欧洲，超速导致的高速碰撞约占20%~40%，表明实际碰撞场景通常超过既定设计阈值。这一差距突显了增强高速碰撞安全措施的迫切需求。

车钩是降低碰撞事故后脱轨风险的关键部件。作为碰撞的第一道防线，其动力行为在列车碰撞响应中起着重要作用，特别是轮对抬升量。虽然包括车钩、防爬器和车体端部压溃区的顺序激活在内的多级碰撞能量管理(CEM)策略，在25km/h范围内有效保证了结构完整性和乘客安全，但车钩在高速碰撞下的复杂变形和失稳机制尚未完全表征。理解这些机制对预测和防止脱轨至关重要。

研究方法

本研究采用理论建模、数值仿真和参数优化相结合的方法：

  1. 速度敏感型车钩建模：

    • 分析车钩内置粘弹性阻尼器的特性，包括活塞杆、缸体和密封组件
    • 基于缝隙层流、薄孔流动和材料可压缩性建立数学模型
    • 将阻尼器等效为变刚度弹簧和速度相关阻尼器的并联系统
    • 通过台车试验验证模型可靠性，峰值力误差小于5.3%

  2. 车钩非线性元件建模：

    • 膨胀压溃管：忽略初始峰值力，仅考虑加载和稳态力阶段
    • 回转机构：视为非线性扭转弹簧，超过限值时表现为刚性冲击
    • 过载保护装置：中间车钩通常不设置过载保护装置

  3. 列车碰撞有限元模型：

    • 建立四编组列车碰撞有限元模型，包括主动列车(A1-A4)和被动列车(B1-B4)
    • 使用多个非线性离散梁单元模拟阻尼器、膨胀压溃管和过载保护装置
    • 球铰单元和非线性扭转弹簧单元模拟车钩回转机构
    • 转向架分为动力转向架和牵引转向架，采用不同建模策略

  4. 仿真工况设置：

    • 碰撞速度：8、15、25、48、60 km/h五个等级
    • 仿真条件：四编组运动列车撞击静止无制动四编组列车
    • 分析车钩响应、轮对抬升量和脱轨风险

主要结论

  1. 车钩响应特性：

    • 随着碰撞速度增加，各界面车钩动力响应呈现明显梯度特征
    • 低速条件下车钩力缓慢上升，中高速条件下过载保护装置触发
    • 速度增加缩短头车车钩作用时间，加剧中间车钩界面载荷
    • 验证了碰撞能量沿列车纵向传递的滞后特性和结构失效的时间序列依赖性

  2. 轮对抬升规律：

    • 各车辆轮对抬升量随碰撞速度增加而显著增加
    • 60km/h高速碰撞时所有车辆抬升量达到峰值，其中A2车辆最为显著(45.6mm)
    • 25km/h碰撞时各车辆抬升量降至15mm以下
    • 8-15km/h低速范围内所有车辆抬升量收敛至2.0~3.7mm范围

  3. 参数优化效果：

    • 膨胀压溃管稳态力和行程对车辆脱轨风险的影响呈非线性
    • 行程增加使轮对最大抬升量先下降后上升，存在特定最优值
    • “大稳态力-短行程"或"小稳态力-长行程"的参数组合可有效抑制轮对抬升
    • 随着稳态力逐渐增加，对应最低轮对抬升时的压溃管行程逐渐减小

  4. 工程应用价值：

    • 建立的速度敏感型车钩粘弹性动力学模型可靠性高(峰值力误差5.3%)
    • 为优化列车被动安全设计提供理论支持
    • 对降低不同碰撞速度下的脱轨风险具有重要指导意义
    • 为多车钩耦合机制和极端条件下的模型适用性验证奠定基础

文章亮点或创新点

  1. 理论创新：建立考虑薄孔流动、层流特性和粘弹性材料可压缩性的速度敏感型车钩模型
  2. 仿真验证：通过台车试验验证模型可靠性，峰值力误差小于5.3%
  3. 规律发现：揭示车钩响应的速度梯度特性和轮对抬升的非线性规律
  4. 参数优化：提出"大稳态力-短行程"或"小稳态力-长行程"的优化组合
  5. 工程指导：为列车被动安全设计和脱轨风险控制提供理论依据
