摘要

轮对多边形磨耗（WPW）在重载机车中常见，对运行安全构成重大威胁，并急剧降低车轮使用寿命。大多数现有的WPW抑制方法主要关注机械振动，而忽略了牵引传动系统的机电耦合（EC）相互作用。因此，这些方法无法全面且从根本上遏制WPW的发展。为了有效缓解这一缺点，本研究开发了一种基于电流补偿的改进磁场定向控制（IFOC）方法，考虑了牵引传动系统的EC特性。随后，建立了WPW演化模型，将EC模型与IFOC和车轮磨损函数相结合，以评估所提出策略在抑制附加谐波和WPW发展方面的有效性。研究结果表明，所提出的方法有效抑制了由WPW引起的牵引传动系统中的谐波电流和转矩，从而显著减缓了WPW的演化。本研究为缓解WPW提供了新的视角，并在工程应用方面具有良好前景。

关键词

#改进磁场定向控制 #轮对多边形磨耗 #机电耦合 #牵引传动系统

背景与动机

近年来，严重的轮对多边形磨耗已成为中国大功率重载电力机车的一个突出问题。这种现象在轮轨系统和关键机车部件中诱导异常振动，显著加速了某些部件的疲劳失效。WPW的形成机制和演化特征在不同的服役场景中差异很大，因为它们受到复杂运行条件和多因素耦合的影响。迄今为止，尚未建立根本有效的解决方案来消除WPW。

现有方法主要包括：

  1. 改善车轮镟修质量/优化车轮结构：初始车轮不规则性可诱导轮轨系统振动，进而触发WPW形成
  2. 变速运行/路线切换：通过定期调整列车运行速度，避免在特定频率下长期振动
  3. 优化车辆-轨道系统参数：通过优化轮轨耦合系统的结构参数，减少轮轨相互作用的振动幅度
  4. 增强轮轨粘着：改善轮轨之间的粘着可以有效增强接触条件，减少滑动摩擦和缓解车轮磨损

然而，大多数这些方法专注于磨损后处理，通常在WPW已经发生后应用。虽然优化车轮结构参数和增强材料性能可以有效减缓磨损进展，但其高成本限制了在服役车辆中的广泛应用。此外，改善轮轨粘着只能部分缓解磨损演化，但仍未能从根本上消除多边形磨损问题。

更重要的是，现有方法主要关注机械方面，而忽略了牵引传动系统是一个机电耦合系统的事实。WPW可以在电力牵引驱动系统中诱导附加谐波和谐波转矩，这些谐波反过来作用于轮对并加速WPW的发展。

研究方法

本研究从机电耦合角度出发，开发了一种创新的WPW抑制方法，具体包括：

  1. 机电耦合机制分析：

    • 分析WPW如何通过齿轮传动系统以速度波动形式传播到电机转子
    • 研究WPW诱导的谐波在牵引驱动系统中的传播路径
    • 揭示谐波电流和谐波转矩如何反过来加速WPW发展

  2. 改进磁场定向控制策略：

    • 提出基于T轴电流补偿的IFOC方法
    • 将转子速度波动转换为所需的补偿电流
    • 通过PI调节有效抑制T轴电流振荡
    • 采用ARIMA模型预测多边形激励下的角频率

  3. 补偿电流计算算法：

    • 使用自回归积分移动平均模型预测稳态转子速度
    • 计算转子速度波动并将其转换为补偿电流
    • 实现实时电流补偿以抑制谐波产生

  4. WPW演化建模：

    • 在Matlab/Simulink中建立电力牵引驱动系统模型
    • 在Simpack中构建机车-轨道耦合动力学系统
    • 基于Archard磨损模型建立车轮磨损模型
    • 通过SIMAT接口实现机电子系统之间的实时数据交换

  5. 仿真验证方案：

    • 使用19阶理想车轮多边形轮廓模拟WPW
    • 设置机车以75 km/h速度运行35秒
    • 比较传统FOC和改进IFOC的控制效果
    • 通过15次迭代计算验证WPW演化抑制效果

主要结论

  1. 机电耦合机制阐明：

    • 揭示了WPW通过垂直轮轨振动传播到电机转子的路径
    • 证明了WPW诱导的谐波电流可以进一步加速其自身发展
    • 阐明了谐波与WPW发展之间的激励-反馈机制

  2. IFOC方法效果显著：

    • T轴电流波动幅度显著降低约62%
    • 频域分析显示与WPW激励频率相关的谐波分量大幅衰减
    • 定子电流中WPW相关的谐波分量得到有效抑制
    • 对应于WPW的谐波转矩分量显著减少

  3. WPW演化抑制明显：

    • 应用抑制策略后，相同运行里程下的车轮磨损显著减少
    • 15次迭代后，无抑制策略的平均磨损达到2.540 mm
    • 应用抑制策略后，平均磨损降至仅0.476 mm，改善效果显著
    • 特别是在抑制高阶WPW方面表现出强大的性能

  4. 工程应用价值突出：

    • 提供了从机电耦合角度解决WPW问题的新思路
    • 方法具有实际工程应用的可行性和有效性
    • 可以显著延长车轮使用寿命，提高运行安全性
    • 为重载机车的维护和运营提供了技术支持

  5. 技术创新性强：

    • 首次从机电耦合角度研究WPW抑制问题
    • 创新性地将ARIMA模型应用于电机控制策略
    • 建立了完整的WPW演化模型进行验证
    • 为类似工程问题提供了新的解决思路

文章亮点或创新点

  1. 机电耦合视角创新：首次从牵引传动系统机电耦合角度研究WPW问题，突破了传统机械方法的局限
  2. IFOC控制策略创新：提出基于T轴电流补偿的改进磁场定向控制方法，有效抑制谐波产生
  3. ARIMA模型应用创新：创造性地将自回归积分移动平均模型应用于电机速度预测和补偿计算
  4. 完整验证体系：建立了包含机电耦合模型和磨损模型的完整WPW演化验证体系
  5. 工程实用性强：方法具有明确的工程应用前景，可以显著改善重载机车的运行安全性
