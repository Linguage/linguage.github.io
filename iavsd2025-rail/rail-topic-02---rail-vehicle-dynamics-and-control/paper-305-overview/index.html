  • 英文标题：Data-Driven Differentiable Adaptive Ensemble Kalman Filter for Enhanced Vehicle State Estimation
  • 中文标题：数据驱动可微分自适应集合卡尔曼滤波器用于增强车辆状态估计
  • 作者及单位：Maozhenning Yang¹,², Jiarui Hu¹,², Kaiying Zhang¹,², Xiaoguang Ma¹,³, Yuanjin Ji¹,²*, Lihui Ren¹,² (¹同济大学交通运输学院，中国；②上海铁路基础设施耐久性与系统安全上海市重点实验室，中国；③中车南京浦镇车辆有限公司，中国)

摘要

跨座式单轨车辆在城市轨道交通中具有显著优势，如高机动性和对复杂地形的强适应性。然而，这些车辆在运行过程中会受到严重的冲击载荷，特别是在梁连接点，这会导致高应力集中并引起关键部件的疲劳损伤。本研究提出了一种数据驱动可微分自适应集合卡尔曼滤波器(DDAEKF)，用于单轨转向架的实时状态估计。通过将递归贝叶斯滤波与基于深度学习的状态转移和观测模型相结合，所提出的方法提高了动态运行条件下的状态估计准确性，为后续故障诊断奠定了基础。该模型通过数值仿真和1:8比例试验台进行验证，在2000秒评估期间实现了对车辆状态的稳定鲁棒跟踪。结果表明，DDAEKF能够有效估计位移和速度，同时减轻方向估计中的不确定性。该方法为提高跨座式单轨系统的安全性和运行效率提供了可靠的状态监测方法。

背景与动机

跨座式单轨车辆因其独特的优势，如小转弯半径和对复杂地形的强适应性，已被广泛应用于城市轨道交通系统。然而，这些车辆面临重大的工程挑战，其中最突出的是从轨道传递到转向架的冲击载荷。这种冲击在梁连接点处尤为严重，常常导致严重的应力集中。这些集中应力是行走轮和导向轮等关键部件疲劳和冲击损伤的主要原因。

如图1所示，在单轨混凝土梁连接处设计了伸缩缝以适应热膨胀。当车辆通过这些接缝时，车轮上产生的冲击会导致轮毂损坏。这不仅损害了单轨系统的运行安全性和可靠性，还显著增加了维护成本。因此，进行详细的车辆状态估计对于全面了解疲劳失效机制至关重要。

卡尔曼滤波器(KF)作为状态估计的经典算法，已广泛应用于交通系统、机器人导航、目标跟踪和工业自动化。在线性高斯系统假设下，KF提供最优状态估计。然而，在处理非线性系统结构、异构多传感器观测和时变噪声特性时，其建模能力和适应性受到显著限制。

为提高在非线性环境中的估计准确性和鲁棒性，已开发了各种改进的滤波方法，包括扩展卡尔曼滤波器(EKF)、无迹卡尔曼滤波器(UKF)和集合卡尔曼滤波器(EnKF)。EnKF是一种顺序蒙特卡罗方法，是传统卡尔曼滤波器的替代方案，以及基于伴随或四维变分方法。它特别适合解决大状态空间和估计误差的非线性演化。

然而，EnKF的性能仍然高度依赖于过程和观测模型的准确性。在实际工程应用中，环境干扰、建模偏差和传感器故障通常导致状态转移和观测的不准确表示，导致估计误差的显著累积。最近，随着深度学习在时间序列建模和动力系统识别中的快速发展，研究人员探索了数据驱动技术与卡尔曼滤波的集成，旨在构建端到端可训练的状态估计框架。

研究方法

本文采用理论建模、算法开发和实验验证相结合的方法：

  1. DDAEKF模型设计：将DDAEKF公式化为EnKF的扩展，同时保持其核心算法步骤不变。具体来说，使用动态集合成员来表示状态分布。集合数量通过动态调整策略确定：N_ensemble = N_min + ασ_t，其中N_ensemble是集合成员数量，N_min是最小集合成员数量，α是动态调整系数，σ_t是时间t的状态估计协方差的范数或主特征值。

  2. 预测步骤：首先使用先验状态x_{t-1}生成扰动样本集合X_{t-1} = {x_{t-1}^{(1)},…,x_{t-1}^{(E)}}。然后通过非线性状态转移模型f_θ(由多层感知器表示)传播每个集合成员，并添加过程噪声v_t ~ N(0,q_ψ(x_{t-1}))以捕获动态不确定性。

  3. 观测建模：采用随机传感器模型生成粒子级观测值y_t^{(i)} ~ ζ(y_t|y_t)，模拟实际传感器读数的随机性。同时学习测量噪声模型以估计测量噪声的协方差，增强在噪声条件下的鲁棒性。

  4. 更新步骤：将每个预测状态x_t^{(i)}通过观测模型h_ψ(也由MLP表示)得到预测测量值ŷ_t^{(i)}。这些预测与观测数据比较，计算创新协方差矩阵S_t，捕获预测和实际观测之间的差异。

  5. 卡尔曼增益计算：使用集合平均偏差A_t和观测预测偏差HA_t计算卡尔曼增益K_t，这是平衡模型预测和传感器测量影响的关键项。

  6. 集合更新：使用标准卡尔曼更新规则更新每个集合成员：x_t^{(i)} = x̂_t^{(i)} + K_t(y_t^{(i)} - ŷ_t^{(i)})。最终状态估计通过平均校正后的集合获得：x̂_t = (1/E)Σ_{i=1}^E x_t^{(i)}。

  7. 端到端训练：定义三个损失函数：状态预测损失L_fθ、观测预测损失L_sζ和最终估计损失L_final。这些目标指导转移和观测模型的联合优化，使DDAEKF能够从数据中学习，同时保持递归滤波结构。

  8. 实验验证：从动态仿真软件和1:8比例试验台收集转向架的相应加速度和速度数据用于训练。使用PyTorch框架在NVIDIA RTX 3090 GPU上实现模型。

主要结论

  1. 模型有效性：DDAEKF模型在2000秒的长期状态估计实验中表现出色，在动力学仿真软件和1:8试验台中都能实现车辆状态的鲁棒稳定跟踪。

  2. 状态估计精度：所提出的方法能够有效估计位移和速度，同时减轻方向估计中的不确定性。尽管方向向量表现出相对较高的不确定性，该框架仍能随时间准确跟踪整体状态。

  3. 自适应能力：通过动态集合调整和基于深度学习的建模，DDAEKF在保持效率的同时提高了估计准确性。模型能够适应变化的条件，增强稳定性和适应性。

  4. 算法优势：DDAEKF将自适应集合调整与深度学习建模相结合，既保留了传统滤波的结构优势，又显著增强了复杂运行条件下的估计准确性和鲁棒性。

  5. 工程应用价值：该方法为提高跨座式单轨系统的安全性和运行效率提供了可靠的状态监测方法，在预测性维护和故障诊断方面具有潜力。

  6. 技术创新：通过将递归贝叶斯滤波与基于深度学习的状态转移和观测模型相结合，DDAEKF为非线性动力系统中的鲁棒准确状态估计提供了新框架。

  7. 实时性能：模型在2000秒评估期间保持了稳定的计算性能，表明其适用于实时状态监测应用。

文章亮点或创新点

  1. 首次提出数据驱动可微分自适应集合卡尔曼滤波器(DDAEKF)用于单轨车辆状态估计
  2. 开发了动态集合调整策略，能够根据状态估计不确定性自适应调整集合大小
  3. 将深度神经网络与贝叶斯递归估计无缝集成，实现端到端训练
  4. 通过1:8比例试验台验证了算法在实际工程环境中的有效性
  5. 实现了2000秒长期状态估计的稳定鲁棒跟踪
  6. 为单轨系统安全性和运行效率提升提供了新的智能监测方法
  7. 在预测性维护和故障诊断方面具有重要应用潜力
