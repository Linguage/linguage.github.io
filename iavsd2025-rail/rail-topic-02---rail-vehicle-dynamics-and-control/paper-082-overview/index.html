摘要

铁路车辆完整车体的惯性参数是动力学分析和计算的基本参数。由于铁路车辆完整车体是大长宽比、大质量和大体积的对称结构，难以准确建立完整的CAD模型，因此其惯性参数的准确识别一直是工程中的难题。本文首先以铁路车辆完整车体为研究对象，采用四点称重法精确测量质量和水平面内的重心位置；其次，使用专门设计定制的低刚度空气弹簧作为支撑，分别用力锤和激振器进行模态测试，通过分析振动方程应用频响函数质量线法识别车体的惯性参数；考虑工程应用，最后将结果与基于六自由度激励平台的结果进行对比，分析质量线法产生差异的原因。结果表明，采用模态测试质量线法识别铁路车辆完整车体的重心位置和惯性参数是可行的。

关键词

背景与动机

铁路车辆的运行稳定性、乘坐舒适性和安全性等动态性能完全由车辆自身的参数决定，其中完整车体的惯性参数是动态性能优化和分析不可或缺的。由于铁路车辆完整车体是大长宽比、大质量和大体积的对称结构，九个惯性参数相差几个数量级，从而给惯性参数的识别带来相当大的困难。

在工程应用中，转动惯量的测试方法通常分为三类：一是物理摆法，如使用三线摆和扭转振动平台方法测量六个不同姿态的振动周期；二是多刚体动力学法，建立多刚体动力学方程，测试外激励下刚体上相应点的角速度和力；三是振动法，将测试对象安装在弹性元件上，通过激励法测试频响函数，用于结构部件转动惯量的识别。其中，质量线法在汽车和飞机的转动惯量测试中已有许多成功案例。

研究方法

本研究采用实验测试与理论分析相结合的方法：

  1. 四点称重测试：

    • 选择车内架点作为四个支撑点
    • 精确测量车体质量（37,734.11 kg）
    • 确定水平面内重心位置（xG=0.17m, yG=0.02m）

  2. 弹性支撑设计：

    • 使用专门设计定制的低刚度空气弹簧
    • 与圆形橡胶弹簧串联支撑
    • 动态刚度：垂直93.5kN/5mm，横向93.5kN/5mm

  3. 模态测试系统：

    • 数据采集系统：SCR205前端
    • 传感器：10个三向加速度传感器（356A16）
    • 激励设备：力锤（086D50）和电磁激振器（4828-001）各2个
    • 测试参数：带宽256Hz，分辨率0.16Hz，谱线数1600，重复激励2次

  4. 质量线法理论：

    • 建立六自由度振动方程
    • 在高频段存在质量线（系统激励频率远大于刚体模态固有频率）
    • 通过最小二乘法求解九个物理参数

主要结论

  1. 测试方法可行性：

    • 质量线法对重心高度zG、主转动惯量Ixx和Izz识别精度较高
    • 与多刚体动力学方法对比，主惯性参数识别误差在可接受范围内
    • 力锤法和激振器法得到的最高刚体模态分别为1.23Hz和1.20Hz

  2. 频率分离效果：

    • 最低弹性模态频率分别为8.52Hz和8.49Hz
    • 弹性模态与刚体模态频率比达到6.91和7.06
    • 满足质量线法对频率分离的要求

  3. 参数识别精度：

    • 质量参数：与称重结果差异仅0.19%
    • 重心高度：识别精度较高（zG≈1.67-1.70m）
    • 主转动惯量：Ixx误差7.59-6.24%，Iyy误差8.34-11.72%，Izz误差27.28-21.46%
    • 惯性积：相对误差较大，因对称结构惯性积接近零

  4. 工程应用价值：

    • 方法操作简便，设备要求简单
    • 相比传统方法成本更低
    • 适用于铁路车辆车体惯性参数的工程识别

文章亮点或创新点

  1. 方法创新：将质量线法成功应用于铁路车辆完整车体惯性参数识别
  2. 支撑设计：专门设计低刚度空气弹簧支撑系统，实现良好的频率分离
  3. 对比验证：与六自由度平台多刚体动力学方法进行详细对比分析
  4. 误差分析：深入分析各种参数识别误差的产生原因
  5. 工程实用：为铁路车辆车体惯性参数识别提供实用的工程解决方案
